# 가구 티어 1~3 — 선정과 시각 문법 (#157, 디렉터 2026-07-10)

> 원칙: **T3 = 현행 모델 그대로**(아트 재작업 0), T1/T2를 신작. 폐허에서 주워 만든 것(T1) → 낡았지만 물건다운 것(T2) → 온전한 것(T3).
> 진행: T1 저코스트 초반 → 업그레이드는 **그 자리에서 개선**(수거·재제작 아님 — "손질" 테마). 쾌적함·효과 티어 스케일.
> 범위: 84 전부 X, 6 코어만 X → **생활 체감 큰 17종 + 후보 3종** (디렉터 컨펌 라인). 시그니처 8종·사치품(축음기·지구본)·액자류는 단일 티어 유지.

**선정 논리 (UX 체감 순):** ①수면(bed)·난방(stove)·조명(lamp/candle/lantern)·식수(purifier)·식량(fridge)은 매일의 생존 루프에서 손이 가는 곳 — 티어가 곧 "오늘 삶의 질". ②sofa/rug/cushion/curtain은 화면 점유가 커서 방 분위기를 지배 — 업그레이드 순간 방 전체가 달라 보인다. ③radio/teatable/chair/table은 인카운터·아침보고 등 연출 앵커라 자주 시선이 간다.
