## 0. 정체 (불변 축)

- **한 문장** (GD-THESIS): "세상이 끝나도, 삶은 계속 손질할 수 있다." — 회복은 사건이 아니라 습관.
- **캐논 3**: 정착자는 나뿐, 타인은 흐른다 / 사망 없음(무력=런의 완결, 기록은 남는다) / 아홉 번의 겨울=제목이 곧 약속.
- **세계관** (WORLDVIEW): 판데믹→소각→핵겨울. "둘러싸였으나 혼자(Alone, Together)" — 폐허=현대적 고립의 은유.
  접촉 스펙트럼(부재/흔적 → 몸 없는 목소리 → 먼 존재 → 스치는 거래 → 찰나의 긴장 → [2.0] 오프스크린 적대) — 커뮤니티·동거인 절대 금지(고양이만 예외).
- **지속 동기 5계층**: L1 손맛 → L2 계절 리듬 → L3 연간 목표 → L4 9겨울 서사 → L5 애착. "압박은 축적의 조미료지 주식이 아니다."
- **난이도 정본**: 코지(normal) / 생존(hard) / 혹한(hardcore) + 무한(zen)·꾸미기(wallpaper). 언락=겨울 통과(무한 1회·꾸미기 코지 2회).

## 2. 레벨 디자인 — 두 개의 축

### 축 1: 파밍 지역 (리스크·시간·보상 테이블)
| 지역 | 성공률 | 위험 | 특산/역할 |
|---|---|---|---|
| 주거 | 0.80 (캡 0.85) | 저(minor) | 식·수·천·초 — 안전 기본기 |
| 상업 | ~0.6 | 중(sprain) | 배터리 확정·의약 |
| 공업 | ~0.4 | 고(deep, 장갑 권장) | 부품·자재·연료 확정 |
| 슬럼 | 0.25 → 숙련 t3 ≈0.61 (캡 0.65) | 최고 | 도료 상인·희귀 가중 — pull의 시험장 |
| 항만 야적장 | 0.50 | 중 | 일일 편중 부스트(복권) |
| 수산시장 | 0.70 | 저 | 소금·신선(겨울 절반) |
| 리조트 | 0.40 (+눈사태 우회 -15%p·1.5배) | 고 | 사치 물자, 케이블카 후 시간 절반 |
| 도심 중심지 | 0.30 + 조우 0.35 | 최상위(2.0) | 책·의약 정점·네온 안료·총 |
| 금지(검문소→연구동) | 방호복/3겨울 게이트 | 엔드게임 | 기밀 문서·희귀부품 |

3축(성공률·시간·위험)이 대체로 단조 — 안전할수록 빠르고 싸다. 위험 지역의 값은 "자원 더"에서 **"거기에만 있는 것"**(도료 계열·도면·네온·시그니처)으로 재설계됨(REWARD-LOOP 이후).

### 축 2: 셸터 사다리 (successes 화폐, 0→290)
컨테이너(0) → 옥탑(25) → 벙커(45) → 오두막(70) → 버스(95) → 지하철(120) → 온실(145) → 여객선(170) → 등대(200) → 예인선(230) → 관제탑(260) → 로지(290).
셸터 = 거처이자 지리(접근성 보너스: 옥탑=상업·슬럼 +5%p 등) + 루트 앵커(벙커 뒷터널→금지, 지하철→선로→도심, 예인선→얼음낚시/해로, 로지→케이블카→고원).

### 게이트의 종류 (전부 세계관 접착)
성공 횟수(셸터 사다리) · 장비(방호복) · 노동(대형 프로젝트: 선로·케이블카·무전 기지) · 시간(낙진 시계 3겨울) · 히든(더블탭 통로) · [2.0] 관문(세관).

## 3. 리텐션·깊이 설계 (요약)

- **REWARD-LOOP 진단**: 진행 게이트가 전역 successes뿐 → 확률 벽 부재 + 난이도·진행 디커플(안전 스팸이 최적, autoplay 실측 slum 0회) + 어려운 곳 보상이 "자원 더"뿐(테라리아 테스트 FAIL).
- **해법 pull 3종**: ①지역 숙련(실패도 성장, 이주 시 두고 감) ②지역 시그니처 코스매틱(도료·도면·복장) ③내구성 가방(빈손 없음 플로어). push 금지 — 코지 보존.
- **DEPTH 정의**: "깊이 = 같은 것을 다시 마주쳤을 때 지난번보다 더 보이는 정도." 신규 시스템 심사 2문: 새 결정인가 새 숙제인가 / 10번째가 1번째보다 흥미로운가.
- **지식 테크트리**(정식판 트랙): 책 24권, 4갈래(온기·자급·살림·앎) × 3티어 — 유일한 플레이어 역량 세로축.
- **모드 = 압박 다이얼 1개**: 같은 시스템이 코지=성취 / 생존=결정 / 혹한=숙련으로 읽히게(트리 분기 금지).

## 5. 2.0 「응답」 요약

1.4의 송출에 **응답이 돌아온다** — 콘텐츠 확장이 아니라 "이야기를 끝내는 업데이트".
- 무대: 동부 신영토(기존 지도 불변). 세관 관문 → 부산형 항구도시. 셸터 4(펜트하우스·다리 관리소·세관·역 대합실, 심부 진행 세관=다리→역→펜트하우스) + 파밍 8(앵커당 2). 아트=TLOU 감성+붉은 노을, 키워드 "압도".
- 시스템: 낙진 시계(3겨울=동쪽 길) · 적대 다이얼(노말 소리만/하드 마주침/혹한 무총 중상 — 전부 오프스크린) · 총=정비 결정(전투 아님) · 엔딩 3분기(탈출/신세계/안식)+히든 「침묵」(사일로, 무기록) · 도시 시그니처=복장 · 전파망(비동기, 조합형 코드북).
- §9 선행분(낙진·적대·총·깊이 3종·엔딩·침묵)은 gd-2.0에 **구현 완료**. 남은 대형 트랙=동부 영토 그 자체(아트+지역 8+셸터 4+복장 렌더).
- 실행: 2.0-α(세관+지역 2+셸터 1) → β(역·다리+지역 4+복장) → 정식(펜트하우스+지역 8+동부 서사).
