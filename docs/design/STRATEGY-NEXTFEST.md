## 1. 스토리보드 — 서사 아치와 "전달"

### 1.1 정서 관통선
- **"둘러싸였으나 혼자"** (현대적 고립 은유). 플레이어 = 남은 정착자 '나', 고양이 = 유일한 동반, 겨울 = 시간의 경과.
- **핵심 리스크(적대적)**: 「9 winters」의 의미가 아직 **작가의 머릿속**에 있다. 디렉터가 방금 CTO에게 *설명*해야 했다 = 플레이어도 설명 없이는 "루프 9번"으로 느낀다. **숭고(영원회귀) vs 노가다(반복)의 경계는 면도날** — 그 차이는 게임이 반복을 *의미로 프레이밍*하느냐에 달렸다.

### 1.2 아치를 플레이어에게 박는 3개 장치 (전부 앰비언트 — 코지는 컷신이 아니라 조용히 말한다)
1. **겨울 수첩(memoir) 페이지** = 서사 척추. 이미 존재. 매 겨울 = "그 해 무엇이 변했나"의 한 페이지. **여기에 9겨울의 누적 의미를 싣는다** (초반=견딤 → 중반=익숙함/권태 → 후반=선택).
2. **엔딩 분기 3종** (설계 필요, 현재 미구현):
   - **연결(connection)**: 라디오/지하철/다른 도시로 타인에게 닿는다.
   - **초월(transcendence)**: 내면을 파다 홀로 초연해진다.
   - **영원회귀(eternal return)**: 오늘의 코지·고양이를 위해 머무름을 *택한다* (디렉터 개인 철학의 코어). 
   → 셋 다 "플레이어의 누적된 하루"가 만든 결과로 읽혀야 한다. 갑툭튀 선택지 아님.
3. **셸터/도시 이동** = 아치의 물리적 그릇. 2.0 도시들을 9겨울에 걸쳐 오간다 = 각 도시 = 한 장(章). **떠남의 정서적 무게**(정든 거처를 두고 감)가 아치를 밀고 간다.

### 1.3 스토리보드 비트 (플레이 흐름)
- **개막**: 홀로 눈 뜸 → 초봄 첫눈 → 고양이가 곁에 옴 → 창밖 폐허. "세상은 끝났고, 나는 남았다."
- **일상 의식**: 고양이·온기·창·꾸미기의 반복 (여기가 코어 — 반복이 위안이 되게).
- **첫 겨울 도착** (= 데모 클라이맥스, 컷 지점): 첫 한파, 불, 고양이와 견딤.
- **이주**: 첫 거처를 떠남 (정서 피크).
- **9번째 겨울**: 3엔딩 중 하나 = 페이오프.

**결론**: 그릇(2.0 이동 구조)은 옳다. **부족한 건 그릇에 담길 의미의 *전달 설계*** — memoir 카피 아치 + 엔딩 3분기 구현이 서사 최우선 과제.

## 2. 깊이감 — 코지에 맞는 깊이 (시스템 복잡도 ≠ 깊이)

**금지**: 스트레스를 만드는 시스템(전투 복잡도·자원 압박 심화). 그건 코지를 부순다. 난이도는 별도 다이얼로 이미 분리됨.

**추가할 "옳은 깊이" 4축**:
1. **의식(ritual) 깊이**: 의미 있는 하루의 순간 증식 — 고양이 루틴/교감, 계절 의식(첫눈·해빙·수확), OST가 시간대·계절에 반응해 바뀜(에셋 있음 → 배선만).
2. **개인화 깊이**: 거처 = 자기표현. 감정적으로 의미 있는 꾸미기(테마세트·코트·"내 공간이 되어간다"). 이미 강점 — 더 밀 것.
3. **메타/서사 깊이**: 겨울이 쌓이며 의미가 쌓임(memoir·시간 경과감·엔딩을 빚는 선택). §1의 아치.
4. **발견 깊이**: 「그려지는 발견」 지도·로어 메모·라디오 방송 = 세계의 이야기를 천천히 벗김. 문서 희소화(2%)가 이걸 "긴 추적"으로 만듦 — 유지.

**한 줄**: 깊이 = 압박이 아니라 **위안과 의미를 더 깊게**. 중반 권태(경제 flat·루프 반복)를 시스템으로 풀지 말고 **의식·서사·발견의 리듬**으로 풀 것.

## 3. cozy core 요소 감사

코지 장르 기둥 = 안전(징벌적 실패 없음, 하드코어는 옵트인) · 따뜻함(시청각) · 동반(고양이) · 완만한 진행 · 아름다움 · 자율(무압박) · 의식.

| 기둥 | 상태 | 비고 |
|---|---|---|
| 시각적 따뜻함 | ✅ | 라이팅·계절·복셀+수제 아트. 강점. |
| 동반(고양이) | ✅(이번 세션 수복) | "기괴" 얼굴로 배포됐던 게 경고 — 기둥도 깨진 채 나갈 수 있다. |
| OST/오디오 | ◑ | **에셋 완성, 배선 대기.** 디렉터 오더 시 투입. 코어의 절반이니 배선은 데모 전 필수. |
| 완만한 진행 | ✅ | 방치·자동저장·취침 자율. |
| 의식(daily ritual) | ◑ | 루프는 있으나 §2.1로 심화 여지. |
| 동반 교감 깊이 | ◑ | 고양이 귀엽지만 **유대**(갸르릉·루틴·반응)는 더 깊을 수 있음(소스 대기). |

**적대적 한 줄**: 기둥은 대체로 있으나 **"있음"과 "탁월함"은 다르다.** cozy 게임은 기둥 하나가 어중간하면 전체가 어중간해 보인다. 데모 전 = 고양이·OST·첫 5분 warmth를 *탁월*로. 그걸 지키는 건 §7의 골든 스크린샷 게이트.

## 4. 트레일러 스토리보드 (Next Fest, 60~85초)

**원칙**: 코지 트레일러 = 60초 안에 *느낌*을 판다. 전투/스트레스 0컷. OST가 주인공. 텍스트 최소, 순간 최대.

| 시간 | 샷 | 의도 |
|---|---|---|
| 0–5s | 창밖 눈, 램프 불빛, 웅크린 고양이. 텍스트: **"세상은 끝났다. 당신은 남았다."** | 훅 — 폐허+따뜻함의 병치. |
| 5–20s | 일상 몽타주: 눈뜸→고양이→거처 정돈→꾸미기→온기. | 코어(위안의 반복)를 판다. |
| 20–35s | 계절 전환(봄→겨울), 아름다운 폐허·잠식, 「그려지는 발견」 지도. | 세계의 규모·미감. |
| 35–50s | 첫 겨울 도착 — 한파·불·고양이와 견딤. 도시 폐허 탐험(고요). | 긴장이되 코지. 데모의 클라이맥스. |
| 50–65s | 깊이 티저: 여러 셸터, 이주, 라디오(연결), 지하철. | "더 있다" — 정식판 볼륨. |

| 75–85s | CTA: **"Nine Winters · 위시리스트 · 데모 플레이"** + 로고 + 따뜻한 마지막 샷. | 전환. |

**보조 컷(숏폼/틱톡용, 별도)**: 고양이 클립 15초, 계절 타임랩스 15초, "폐허인데 아늑" 반전 15초. 코지+고양이는 숏폼 알고리즘 친화적 — **본 트레일러보다 숏폼이 위시리스트를 더 끌 수 있음.**

## 5. 경쟁작 & 장르 포화도 (정직하게)

**포화 진단**: "cozy"는 마케팅 태그로 **포화**. 그러나 세부 니치(**포스트아포칼립스를 코지로** + 방치 + 고양이 동반 + 웹/모바일 접근성)는 상대적으로 비어 있음.

**비교군(정직)**:
- **거인/직접**: Stardew Valley(코지 농사, 벤치마크 불가능한 거인), Spiritfarer(코지 경영+죽음/정서 — **가장 가까운 톤: 코지+멜랑콜리**), Cozy Grove, Unpacking, A Short Hike, Terra Nil(치유형 재건), Dredge/Dave the Diver(코지-어드벤처).
- **안티-레퍼런스**: **Frostpunk**(하드 생존 — 우리가 *아닌* 것. 마케팅에서 명확히 선 그을 것).
- **방치/모바일 코지**: 다수. 여기서 "웹/모바일 태생"이 양날 — 접근성↑ but "모바일 게임" 인식 = 가격 저항.

**차별점(전면에 세울 것)**:
1. **폐허를 grim이 아니라 warm으로** — "잿더미 속 아늑함"의 반전. 이게 유일한 훅.
2. **고양이 동반** + 멜랑콜리 톤(Spiritfarer 관객).
3. **철학 아치(9 winters)** — 사색적 코지 관객(생각하게 하는 코지)을 겨냥.
4. **접근성** — 웹/모바일/PC 어디서나.

**리스크**: 무명 IP + 포화 태그. 캡슐/트레일러가 훅을 못 세우면 묻힌다. → **"post-apoc but cozy + 고양이 + 멜랑콜리"를 캡슐 1초에 박아야 한다.**

## 6. Next Fest 출품 경쟁력

**현실**: Next Fest = 데모 수백 개. 위시리스트를 얻는 것 = ①강한 첫 5분 ②캡슐/트레일러의 즉각적 훅 ③퀄 신호(폴리시) ④스트리머 친화.

**우리 강점**: 유니크한 톤, 고양이, 접근성, **데모 컷 지점(겨울 도착=감질맛)** 이 전환 설계로 옳음.
**우리 약점**: 무명, 포화 태그, 캡슐/트레일러가 아직 미검증, "모바일 포트" 인식 리스크.

**데모 전 필수 액션(우선순위)**:
1. **첫 5분 폴리시** — 눈뜸→고양이→첫 온기까지가 훅. 여기 warmth를 탁월로(§3).
2. **OST 배선** — 코어 절반. 데모에 반드시.
3. **캡슐 아트 + 트레일러** — "폐허+따뜻함+고양이" 1초 전달.
4. **데모 밸런스 재빌드**(#74) — 현 데모는 구 노말 경제. 최종 밸런스로.
5. **전 모드 오디트**(#90) — 데모 노말만이지만 코어 정합.
6. **데모 종료 화면 CTA** — 위시리스트 링크(이미 있음, 강화).

## 7. 가격 타당성 & 시장성

### 7.1 가격 (정직한 권고)
- 현 **$11.49 정식**은 무명 코지 인디 + 포화 장르 + 웹/모바일 태생 인식에는 **살짝 높다.** $10 위는 "충동 구매" 심리 구간을 벗어나 *신뢰*를 요구하는데, 무명 IP엔 그 신뢰가 아직 없다.
- **권고: $9.99** (10불 미만 = 충동 구간 + "너무 싸지 않음"). 더 공격적 Next Fest 푸시면 **$8.99**.
- **하한 주의**: $5~7은 "모바일 포트/셰어웨어" 신호 위험. 디렉터가 언급한 **7.49~9.9 밴드는 합리적** — 그 안에서 **9.99 앵커, EA/출시 할인으로 실구매가를 낮추는** 전략이 위시리스트 전환에 유리(할인폭이 기본가보다 전환에 더 작동).
- Play $3.99 = 별개 시장, 유지 OK.

### 7.2 시장성
- **강점**: 고양이+코지+폐허 톤 = 숏폼(틱톡/릴스/쇼츠) 친화. 고양이 클립·계절 타임랩스·"폐허인데 아늑" 반전이 알고리즘을 탄다 → **유기적 도달의 주력은 숏폼**.
- **사색적 코지 관객**(Spiritfarer·A Short Hike의 "생각하게 하는 코지") = 철학 아치의 타깃.
- **웹 데모의 이점**: 설치 없이 즉시 플레이 = 위시리스트 마찰 최소. Next Fest Steam 데모와 별개로 **웹 데모 링크를 숏폼/커뮤니티에 뿌릴 수 있음**(경쟁 우위).

## 9. 적대적 최종 경고 3개
