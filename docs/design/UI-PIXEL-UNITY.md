## 1. 정본 규격 (신규/수정 UI가 따르는 계약)

- **픽셀 그리드**: 게임은 `innerWidth / opts.pixel`로 렌더(1텍셀 = `opts.pixel` px). UI 텍스처도 **동일 텍셀 크기**를 목표. 픽셀본은 저해상으로 굽고 `image-rendering: pixelated`로 확대(다운스케일 금지 — 도트가 뭉갬).
- **팔레트**: 공용 한정 팔레트로 스냅. 정본 도구 `tools/icon-pixelize.mjs`(grid24·K32·베이어). 아이콘 세트는 **세트 전체 1회 median-cut**(낱장 팔레트 = 불균질 = AI티). 고채도 액센트(앰버·적색 LED·인광)는 전용 슬롯 보존.
- **폰트**: `DungGeunMo` 전면. 명시 sans-serif/시스템폰트 금지, `font-family: inherit` 강제. 크리스프 사이즈(네이티브 그리드 배수)는 **실화면 검증으로 확정**.
- **보더**: 2px 하드 픽셀 보더. `border-radius ≤ 2px`(LCD/종이 내부는 0). **1px 이하 AA 보더 금지**(창=도트, 버튼=AA 로 쪼개지면 안 됨).
- **image-rendering**: 모든 픽셀아트 표면에 `pixelated`. 스무스가 정당한 곳(초소형 다운스케일 등)만 예외 + 이유 주석.
- **선명도 위계**: 1차 정보(자원·상태)=선명·채도. 2차(장식·배경)=디밈·저채도. 채도 튀는 아이콘이 무드 깨면 팔레트로 낮춘다.
- **텍스트 안전**: 아이콘→텍스트 순, `keep-all`+말줄임/줄바꿈 규칙, 컨테이너 밖 삐짐 0.
