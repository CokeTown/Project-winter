# 일본어(ja) 로컬라이제이션 규약 — #191

> 목적: ko 원문 → ja 번역의 톤·용어 단일 기준. 신규 문자열은 ko+en+ja 3언어가 기본(#i18n-both 확장).
> 원문 기준은 **한국어**(ko). en은 참고용.

## 톤(레지스터) 매핑 — #139 어투 체계의 일본어 대응

| 표면 | 한국어 | 일본어 |
|---|---|---|
| 플레이어 행동·결과 노트(혼잣말 반말) | "물을 마셨다" | 常体 독백체 — 「水を飲んだ」 |
| 시스템 공지·설정·안내(다나까) | "저장되었습니다" | です・ます체 — 「セーブしました」 |
| UI 라벨·버튼 | "저장" | 体言止め 간결형 — 「セーブ」 |
| 라디오·방송(단파 무전 톤) | 단문, 잡음 섞인 톤 | 단문·불완전문 유지, 무전 관용구 「こちら○○」「どうぞ」 |
| NPC 인용 대사("...") | 화자 성격대로 | 화자 성격 유지(2인칭 가능) |
| 서술문 POV(#176) | 1인칭 나/내 | 1인칭 — 주어 생략 우선, 필요 시 「私」 |

- **{josa}는 ko 전용** 플레이스홀더 — ja에선 제거하고 자연스러운 조사(を/が/に…)를 직접 쓴다.
- 그 외 플레이스홀더 {n}·{item} 등은 **원형 그대로 보존**(개수·이름 일치 — check-i18n 게이트).
- 줄바꿈(\n)·마크업(있다면)도 구조 보존.

## 조판·폰트

- `html[lang="ja"]`에서 keep-all 전역 해제 + `line-break: strict`(금칙처리) — style.css.
- 폰트: DungGeunMo(라틴·숫자만 유효) → Meiryo/Yu Gothic UI/Noto Sans JP 시스템 폴백.
  **일본어 픽셀 폰트(PixelMplus 등, M+ 라이선스) 도입은 디렉터 결정 대기** — 도입 시 여기 갱신.
- UI 길이: ja는 ko보다 대체로 길다 — 버튼·라벨은 원문보다 짧거나 같게 의역 우선.
