# 동부 경제 정리 — 기존 안건 + 확정 방향 반영안 (2026-07-17)

> 디렉터 방향(2026-07-17 접수): **"코지는 본토와 전반 파밍 난이도 비슷하게. 하드·하드코어는 좀 더 하드하게 —
> 도심 지역이니 구하기 힘든 게 정상. 대부분 셸터를 강화해서 수급할 수 있도록."**
> 이 문서는 ①흩어져 있던 기존 안건을 한자리에 모으고 ②위 방향을 수치안으로 변환한 것. 수치 컨펌 시 즉시 구현.

## 2. 현재 실값 (world.js — 오늘 기준)

**홈(첫 도시)**: 주거 0.80 · 수산시장 0.70 · 상업 0.60 · 항만 0.50 · 검문소 0.50 · 공업 0.40 · 리조트 0.40 · 연구동 0.35 · 도심 중심지 0.30 · 슬럼 0.25 · 뒷골목 0.22
**동부(항구 대도시)**: 고급 주거단지 0.55 · 세관 압류창고 0.50 · 인터체인지 0.50 · 대승강장 0.45 · 거대상가 0.45 · 컨테이너항 0.40 · 백화점 0.40 · 군사 아웃포스트 0.30

읽는 법: 동부는 **최상급 안전 지역(0.7~0.8대)이 아예 없는 분포**다. 홈의 "주거지역=초반 텃밭" 같은 만만한 곳이 없고, 전부 홈 중반~후반 지역대(0.3~0.55)에 몰려 있다. 즉 **분포 자체가 이미 "도심=만만치 않다"를 말하고 있다** — 초안이 방향과 거의 정렬돼 있었던 것.

## 3. 방향 반영안 (컨펌 대상 수치)

원칙: **rate 테이블(§2)은 그대로 둔다** — 지역 개성은 지역 상수로, 난이도는 모드 노브로. 전부 `BAL.cities` 아래 노브 3개.

### ① 코지(노말·무한) = 본토와 체감 등가 → **무보정**
동부 평균 rate 0.44 vs 홈 평균 0.47 — 이미 등가 밴드. 숙련(masteryBonus)·가방·피티가 홈과 동일하게 얹히므로 코지는 손대지 않는 게 "비슷하게"의 가장 정직한 구현. **eastRateAdj.normal = 0, zen = 0.**

### ② 하드·하드코어 = 동부만 한 겹 더 → 노브 2개
- `eastRateAdj`: **hard −0.06, hardcore −0.10** (동부 지역 한정 성공률 가산 페널티. 슬럼 0.25가 홈 최저인 걸 감안, 아웃포스트 0.30이 hardcore에서 0.20까지 내려가는 정도 — "험하지만 로또는 아닌" 선)
- `eastLootMul`: **hard 0.85, hardcore 0.75** (성공해도 수확이 짠 도심 — 실패율과 수확량을 분리해서 조이면 "갔다 오면 뭔가는 있는데 늘 모자란" 감각이 남습니다. 빈손 방지 가방 플로어는 그대로 작동)

### ③ "셸터를 강화해서 수급" = 설비 산출은 모드 무관 + 동부 4채 설비 확충
설비(텃밭·빗물받이·태양광·버섯재배 등) 산출에는 위 페널티를 **적용하지 않습니다**. 하드에서 파밍이 짜지면 "설비를 갖추는 게 답"이라는 루프가 노브 추가 없이 저절로 성립합니다(흉작 확률 #108은 기존대로).
동부 4채 설비 감사+확충안(각 1~2종, 도심 정체성에 맞게):
- **펜트하우스**: 옥상 온실(텃밭 상위 호환 — 수확 +1) + 기존 태양광 앵커
- **다리 관리소**: 강풍 소형 풍력(태양광 대체 급전 — 흐린 날도 도는 대신 출력 낮음) 〔신규 설비 1종〕
- **세관**: 대형 빗물탱크(빗물받이 상위 — 저장 상한 ↑)
- **역 대합실**: 매점 자판기 복구(격일 통조림/물 소량 — 벙커 문법 재사용)

### ④ cross-city 이주 코스트 (미구현분 마감)
국경을 넘는 이주는 여정 물자를 **음식 2·물 2 + 6시간**(구역 간 1·1·3h의 두 배)으로. 하드 계열 배수는 기존 gateCostMul이 정비 자원에만 걸리는 현 문법 유지(여정 물자는 생존 상수 — #108 결정 존중).

## 4. 구현 계획 (컨펌 후 1유닛)

1. `BAL.cities.eastRateAdj`/`eastLootMul` 노브 + `startExpedition`·`resolveExpedition`의 rate/수확 합산부에 city 분기(rateParts 문법 — 항 1줄).
2. moveCostFor에 cityOf 겹(음식2·물2·6h) + 이주 카드 비용 표기.
3. 동부 4채 설비 4종(③) — 각 buildRoom 앵커+SHELTER_MODS+제작 코스트+3언어.
4. 검증: 모드×동부 소크(코지 체감 등가 밴드 실측 — 홈 대비 수확/일 ±10% 이내, 하드·하드코어 하향 실측) + 배터리 + sim 재캘리브 1회(§9.8.12-e).

## 5. 열린 질문 (같이 컨펌해 주시면 한 번에 갑니다)

- ②의 수치 2쌍(−0.06/−0.10, 0.85/0.75) — 이대로 갈지.
- ③의 신규 설비 4종 콘셉트 — 특히 풍력(신규 설비 1종이라 아트+밸런스 비용이 가장 큼). 빼고 3종으로 시작해도 됩니다.
- 세관 검문 서사(미결 2 잔여): 국경 통과에 일회성 인카운터를 붙일지 — 경제와 별개라 후순위 제안.
