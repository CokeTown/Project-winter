## 1. 왜 "응답"인가 — 1.4가 끝처럼 보이는 문제의 해소

베타 테스터가 1.4를 결말로 느낀다 = 설계대로 읽은 것(1.4는 "엔드게임" 라벨 + 주제 회수까지 완료).
2.0의 위험 = 사족으로 보이는 것. 해법은 레트콘 없이 1.4의 마지막 비트를 **소급해서 문으로** 만드는 것:

- 1.4 결말: 무전 기지 복구 → 수집 방송 송출 → 지도에 생존자 불빛 점등. *(현재는 여기서 '답'으로 끝남)*
- **2.0 개막: 그 송출에 응답이 돌아온다.** 지도의 불빛들 = 이제 갈 수 있는 곳. "끝인 줄 알았던 것이 초대장이었다."
- 마케팅 포지션: 콘텐츠 확장이 아니라 **"이야기를 끝내는 업데이트"**. 1.x = 세계의 완성, 2.0 = 서사의 완성.

## 2. 낙진 시계 — 시간이 세계를 연다 (디렉터 확정)

핵겨울 낙진은 2~3년이면 상당 부분 가라앉는다(현실 근거 있는 밴드). 이를 게임 시계에 박는다:

- **1~3번째 겨울**: 금지 구역(1.4)은 방호복으로만. 도심 중심지는 진입 불가(치사 선량).
- **3번째 겨울 이후**: 낙진이 걷혀 **금지 구역이 맨몸 개방**되고, 그 너머 **도심 중심지(신규 최상위 지역)**가 열린다.
- **방호복 재프레임** (가치 붕괴 방지): 방호복 = "먼저 들어가는 자의 어드밴티지". 1~3겨울에 선진입 하이리스크 루팅
  vs 3겨울 후 안전 진입의 후발 — 투자 시점이 결정이 된다. (지역 숙련②와 시너지: 먼저 들어간 자는 숙련도 먼저 쌓는다.)
- 깊이 정의와의 정합: "같은 지도를 다시 마주쳤을 때 더 보인다" — 시간이 지도 자체를 바꾼다.

## 3. 도심 중심지 — 하이 밸류, 하이 텐션 (디렉터 시나리오)

- **루팅**: 하이 밸류 물건(희귀 부품·의약 정점·사치 가구·책 뭉치)의 최상위 소스. 후반 경제의 새 천장.
- **밴딧 (약탈자 무리)**: 낙진이 걷히자 사람들이 심장부로 몰린다 — 그리고 "함께 살지 않는 이유"가 실증된다(§2 저류의 물리화).
  - 수위 규칙: **오프스크린/간접 우선** — 통행세 쪽지, 경고 사격 *소리*, 털린 은닉처, 모닥불 연기. 대면은 희귀 + "찰나의 긴장 접촉" 층위(회피/거래/손실의 결정, 온스크린 전투 시스템 없음).
- **변종 (방사능 변이체)**: 판데믹 병원체가 방사능 환경에서 변이한 무언가 — 도심 깊은 곳에서 *목격되는* 존재.
  - 수위 규칙: **크리처 호러 금지. 스토커의 존(Zone) 문법** — 소리, 그림자, 스쳐 지나감, 흔적. "봤지만 싸우지 않았다."
  - 서사 기능: 공포 장치가 아니라 **연구실(§4)로 이끄는 빵부스러기**.
- **적대 존재 모드 다이얼 (디렉터 확정 2026-07-08)**:

  | 모드 | 적대 존재의 실체 |
  |---|---|
  | 노말 | **소리만 낸다** — 존재는 앰비언스와 소문. 습격·조우·손실 없음 |
  | 하드 | **마주친다** — 조우 시 파밍 실패 가능성 발생 (전리품 손실/빈손 귀환) |
  | 하드코어 | **총이 없으면 탐사 중 중상을 입을 수 있다** (사망 아님 — "사망 없음, 탈진만" 정체성 유지. 디렉터 확정 2026-07-08). **중상 = 부상 서사화(④)의 정점 티어**: 의약품 소요 급증·회복 장기화·memoir 흉터. 총 = 내구성제(소모), 연구실에서 파밍, 수리 가능(코스트 비쌈). 발사는 오프스크린 판정 — 전투가 아니라 로드아웃·정비의 결정 (DEPTH-DESIGN §10.2) |

## 4. 진실 — 연구실과 다음 인류 (디렉터 시나리오)

도심 중심지의 심부에 **연구실**이 있다. **그들의 정체와 의도 (디렉터 확정 2026-07-08)**:

- 소각을 결정한 이들 중 일부는 **상호 확증 파괴를 멈추고**, 새로운 세계를 지으러 떠났다 — 그곳의 잔존 인류를 모아 다시 신세계를 만들려 한다.
- 단, 그것은 우리가 바라는 신세계(모든 인류가 하나 되어 재건·화합·규합)가 아니라 — **기술을 가진 자신들의 주도로 설계한 신세계 질서**다.
- **그들은 살아남는 자를 지켜봤다. 이곳에서 버텨낸 주인공은 그들의 눈에 '합격'했다.** — 이 사실은 연구소와 봉쇄 구역들의 메모로만 발견되도록 제작한다 (조각 순서: 관찰 기록 → 선별 기준 → 규합 계획 → 초대).
- 기존 스파인과의 접합: 박사(소각 반대표, rsc9)와 궤도 관측소는 이 네트워크의 마지막 관측자. 변종은 이 진실의 그림자.
- lore.js rsc 라인의 종착지: "소각은 끝이 아니라 이관(移管)이었다" — 세계관의 답(#35)의 최종 층. 신세계 엔딩의 양가성(§5)은 여기서 나온다: 인류의 재건이라는 도약 / 거절할 수 없는 초대라는 순응.

## 5. 엔딩 3분기 — 재정의 (디렉터 언어로 갱신)

구 정의(연결/초월/영원회귀)를 디렉터 시나리오로 **구체화**. 정서 매핑은 유지, 그릇이 선명해졌다:

| 엔딩 | 무엇 | 시점 | 구 매핑 |
|---|---|---|---|
| **탈출** | 진실을 다 보기 전에 박사의 연락을 받고 구조선에 오른다 — 떠나는 자의 엔딩 | **조기 가능** (9겨울 전, 박사 스파인 진행 시) | 연결의 원형 (기존 구조 엔딩 흡수) |
| **신세계** | 연구실의 진실을 받아들이고 다음 인류의 질서를 향해 나아간다 | 아홉 번째 겨울 | 연결의 어두운 구체화 |
| **안식** | 진실을 알고도 — 오늘의 고양이, 이 집, 나의 겨울을 위해 다시 숨는다 | 아홉 번째 겨울 | 초월+영원회귀 통합 (디렉터 철학의 코어) |
| **(히든) 침묵** *(가칭)* | §5.1 히든 루트의 끝 — 사일로의 버튼. 그러고 끝 | 히든 루트 완주 시 | 4번째 축: 거절마저 거절 |

- **신세계의 온도 (디렉터 확정 2026-07-08)**: "다시 인간들의 세상"을 만들기 위한 **도약**임과 *동시에*, 거절할 수 없는 **"섬뜩한 순응"**. 희망과 공포를 둘 다 — 어느 한쪽으로 해석을 닫지 않는다. lore 저작은 이 양가성을 유지할 것.
- **Day 10000** = 어느 엔딩이든 그 후의 먼 에필로그(관측 위성 다리 서사) — 계속 플레이 보상으로 존속.
- 엔딩은 "누적된 하루"가 빚는다: 어디서 겨울을 보냈나(도시), 무엇을 주웠나(진실 조각), 무엇을 지었나(집) — 갑툭튀 선택지 금지(STRATEGY §1.2).

### 5.1 히든 루트 — 올라가는 곳이 없는 지하철 (디렉터 확정 2026-07-08, 진짜 이스터에그)

발견 자체가 콘텐츠인 4번째 길. **어떤 UI 힌트도 없다** — 커뮤니티가 서로에게 속삭이며 발견하는 종류의 것.

1. **입구**: 지하철 선로 복구(1.2)를 **끝까지** 완성하면 노선의 끝에 연결 지점이 생긴다 — *올라가는 곳이 없는* 역(1.4 구현 그대로 활용 — 이미 시각적으로 출구가 없다). 승강장 어딘가의 **특정 지점을 터치/클릭**하면 특수 통로가 열린다 (숨겨진 인터랙션 — 눌리지만 아무 표시가 없다).
2. **개척**: 통로 너머는 **막대한 자원**을 들여 개척해야 한다(대형 프로젝트 최고 코스트). 완주하면 **2.0 지도로 이어지는 네 번째 문** — 정규 3수단(지하철·케이블카·예인선)과 별개의 비밀 경로.
3. **연구소와 유보**: 이 길로 연구소에 닿아 **박사와 상호작용**한다. 떠나지 않고 남은 채로 — 선택을 **유보**한다.
4. **박사의 문서**: 유보한 자만이 발견한다 — 박사가 남긴 문서. **그들이 어디로 갔는지, 그리고 미사일이 어디로 갈 수 있는지**가 적혀 있다. 그들의 의중(§4 — 선별·규합·자기들의 질서)을 이미 메모로 읽어온 플레이어에게, 이 문서는 좌표가 된다.
5. **사일로**: **미사일 사일로**. 버튼을 누르면, 그들이 간 곳으로 핵이 터진다. **그러고 끝.**

- 톤: 설명도 정당화도 없다. 상호 확증 파괴를 *멈춘* 자들을, 그 파괴로 끝내는 플레이어 — 게임은 판단하지 않고 끝난다. (침묵이라는 가칭은 여기서 온다.)
- **구현 함의 (디렉터 확정 2026-07-08)**:
  - **지하철 역사 연출**: 1.4 모델 그대로 쓰되 **조명을 확 줄인다** — 버려진 역이므로 어둡게, 남은 조명은 **붉은 비상등만**. (조명값 변경이라 골든 스크린샷 갱신 필요 — 신규 지오는 아님)
  - **계단/사다리는 개척 *후*에 나타난다**: 히든 통로를 제작·개척한 뒤 기존 역사에 계단(또는 사다리) 모델이 추가되어 "연결되어 있다"를 시사한다. 발견 전에는 아무 단서도 없다.
  - 히든 터치 포인트: 시각 변화 없는 순수 히트 영역(골든 게이트 무충돌). 실수 방지로 더블 터치/홀드 요구 검토.
  - 사일로 시퀀스: 신규 아트 최소로 — 어둠·계기판·박사의 문서(종이 연출 재사용)·버튼·화이트아웃이면 충분하다.

## 6. 도시와 이동 — 네 도시의 아홉 겨울 (정본, 디렉터 확정 2026-07-09)

> **⚠️ 디렉터 정정 (2026-07-09 2차 — 아래 전체에 우선)**: 4도시는 **1.4까지의 본편 지도 재편이 아니라 신규 영토**다 — *"신규 구역은 사실상 본편의 2편."* 현 지도(잿빛 외곽~금지 구역)는 **불변**. 확장은 **현 지도의 동쪽**으로 나가며, 관문은 **세관(稅關)** — 국경 검문의 잔해. 1호 신도시 = **부산형 항구도시**(세관 너머의 항만 대도시, 가칭). 도심 중심지는 낙진이 걷힌 뒤 **거주지역화 확정**(구 미결 #8 해소). 이에 따라 §9.8.3의 "기존 10지역→8 폴드"는 **폐기** — 기존 콘텐츠 무손실 + 이 계획의 최대 리스크였던 첫 도시 sim 재캘리브 통째 소멸. 파밍 8곳은 **전부 신규**(신 구역당 2). 엔지니어링 척추(cityOf 파생·이월 3층·엔딩 체류 가중·세이브 가산)는 신규 영토 모델에서도 그대로 유효 — 지리 모델만 바뀐다. §6.1·§9.8.2~4의 구 도시 표·폴드 표는 정정 전 산출물로, 참고용으로만 남긴다.

이동 수단은 1.x가 이미 지었다. 2.0은 그 문을 여는 것 — 그리고 각 도시는 **그곳에만 있는, 갖고 싶은 것**으로 끈다(pull, push 금지 — REWARD-LOOP 테제).

### 6.2 아홉 겨울의 감정 아크

- **W1~2 (첫 도시 고정) — 정 붙이기.** 루프를 배우고, 첫 집을 커스텀하고, 첫 시그니처 가구·도료를 얻는다. "떠남의 무게는 정든 다음에만 성립" — 뿌리내리기 전엔 못 떠난다. 낙진 시계(§2)가 3겨울을 향해 똑딱인다.
- **W3 (첫 이주 창) — 낙진이 걷힌다.** 지하철·케이블카·예인선이 이제 어딘가로 이어진다. 첫 결정: 머문다(안식 씨앗) or 떠난다(설렘). 도심으로 가면 도심 중심지가 열리고(winters≥3) 진실 스파인이 시작되지만 **지도가 백지** — 뒷골목 0.30이 숙련 없이 다시 무섭다. **마스터한 집을 두고 온 무게가 연출 없이 메커니즘에서 온다.**
- **W3~8 (자유 순서) — 중반.** 새 도시마다 백지 지도 + 새 시그니처 복장 + 새 톤. 귀향하면 마스터한 집 지역을 다시 돌고, **집은 픽셀 그대로 나를 기다린다.** 도시 복장을 하나씩 모은다. 전파망(§6.5)이 도시마다 다르게 닿는다. 대한파 프론트(§9.4)가 능선·도심에서 더 매섭다.
- **W9 (어디든) — 구조가 온다.** 어디 있든 9겨울 `ending_choice`가 발화. 누적 도시 체류 + **아홉 번째 겨울의 체류지**가 leaning을 가중하되 강제 안 함(3택+보류 항상 개방).

감정 스루라인: **귀향(뿌리) → 떠남의 무게(숙련 두고) → 백지 지도의 설렘 → 귀향(집 그대로) → 아홉 번째 겨울(어디서 끝났나 = 내가 누구인가).**

### 6.3 이주 리듬

"한 겨울, 한 도시." 이주 = 대형 프로젝트(경로 개척 + 짐, 1.1 시스템 재사용). 왕복 가능하되 비싸게. 계절 창 × 대한파 프론트가 "언제 떠나느냐"를 결정으로 만든다. **9겨울 배분**: 1~2겨울 첫 도시 고정 → 자유 순서 → 9번째 겨울 어디서든 구조. 안 떠난 자의 9겨울도 유효(안식 루트).

### 6.4 도시 시그니처 — 복장을 축으로, 네 벌이 하나의 서사

디렉터 확정: **도심 계열의 시그니처 = 입는 복장.** 이를 네 도시 전체의 **휴대 가능한 도시 정체성 토큰**으로 승격한다 — 각 도시가 자기 복장 한 벌을 주고, **네 벌을 다 모으면 = 모든 도시를 조금씩 집으로 데려온 것.** 기존 시그니처 가구 8종·도료 계열은 불변, 복장은 그 위에 얹는 층이다. 도심이 이 축의 기함(연구원 코트·수도 정장 = 가장 풍성한 옷장)이고, 나머지 도시는 복장 + 각자의 부가 시그니처를 함께 얹는다(§9.8.9).

**획득 = 첫 도착 마일스톤 + 의상 도면(플로어 보장 드랍) + 제작 1회.** 숙련 티어 게이트가 **아니다** — 한 겨울 원정으로 도시 정체성 토큰을 손에 넣을 수 있어야 "한 겨울 한 도시" 케이던스와 양립한다. 숙련 티어는 **복장 변주(리컬러·부가 조각)**만 게이트해 재방문 깊이를 준다. pull은 도시 진입점에 **front-load**, 그라인드 뒤로 back-load 하지 않는다. 전부 파워 0 순수 치장(코지 안전선).

## 6.5 전파망 — 「응답」의 문자적 구현 (디렉터 2026-07-08)

§1의 "응답"은 지금 **스크립트**(먼 불빛·박사)다. 디렉터 제안 = 그 응답 층에 **진짜 다른 플레이어**를 얹는다: 혼자 보낸 전파가 낯선 생존자에게 닿고, 그의 전파가 내 라디오에 뜬다 — *"누군가가 ○○라고 보냈던데, 수신양호."* "둘러싸였으나 혼자"([[WORLDVIEW.md]])의 은유를 메커니즘으로 직역한 것 — **이름값(응답)을 하는 기능.** 기믹이 아니라 2.0의 테제. (정본화: docs/design/REWARD-LOOP.md의 "완성·수집" pull과도 접합 — 수신 자체가 수집 대상이 될 수 있다.)

**형태 — 비동기 유리병 편지 (실시간 멀티 아님)**:
- 플레이어가 전파 송출 → **익명 풀에 POST**. 접속/특정 조건에서 풀의 전파 1개 GET → 라디오 인카운터로 수신. "수신양호/불량"의 라디오 텍스처로 노출을 큐레이션.
- 계정·PII 없음. **오프라인이면 기존 스크립트 풀로 폴백**(코지 오프라인 정체성 사수).

**안전 설계 — 조합형(권장), 자유입력 아님**:
- 유저 텍스트를 낯선 타인에게 = 공개 콘텐츠 → 검열 책임 발생. **큐레이션 문구 조각(무전 코드북)에서 조립** → 출력 공간 유한 → **런타임 검열 불필요** + 단파 무전 톤([#82 방송 전문 단문화])과 정합. 자유입력은 욕설필터·신고·수동검수 부담이라 코지에 과함 → **비채택.**

**2단계 롤아웃 (비용 의식 — 디렉터 "또 다른 코스트는 싫다")**:
- **1단계 (1.x, 백엔드 0)**: 라디오 생존자 풀에 **큐레이션된 실제 전파를 씨앗으로 프리베이크**(베타/데모 폼 수집 또는 저작). 인프라·런타임 검열 0, 감정 페이오프의 8할. `survivorLights`/`distantLight`/`tryRadioBroadcast` 파이프라인 ♻️ 통째 재사용.
- **2단계 (2.0 라이브 릴레이)**: 서버리스(Cloudflare Workers + KV급) async 릴레이 — 익명 POST/GET. 백엔드+검열 = **의도된 비용 결정.** 오프라인-우선 아키텍처를 처음 깨는 지점 → 착수 전 별도 승인.

**정체성 정합**: 수신 = 따뜻한 흔적("바깥에 나 말고도 버티는 누군가")→ 세계관 희망 층. 위협 층은 §3 적대(밴딧·변종)가 전담. **관계 발전·채팅 금지**(접촉 스펙트럼 "찰나의 접촉" 유지).

**상태**: 방향 정본화(디렉터 2026-07-08). **코드 미착수** — 1단계는 1.x에서 저비용 파일럿 가능, 2단계(라이브)는 비용·검열 결정 후.

#### 9.8.1 아키텍처 — 도시는 파생, 저장 아님

현 아키텍처는 이미 2단 파생: `districtOf(shelterId)`(expedition.js:26, export)가 셸터→구역, `moveCostFor`(game.js:2306)가 `districtOf` 불일치로 구역 간 코스트 스케일. 도시는 그 위 한 층:

```js
// world.js — 각 DISTRICT에 city 태그 1개 (역인덱스 CITY_OF 파생)
outskirts·city·meadow·forest·coast → 'home'
research  → 'capital'      // 낙진 후 거주 개방(현 shelters:[] → 신규 셸터 2채 편입)
highland  → 'observatory'
harbor    → 'port'
export function cityOf(id){ return CITY_OF[districtOf(id)]; }   // 무저장 파생
export const currentCity = () => cityOf(state.current);          // 매 로드 시 파생
```

`state.current`은 이미 저장·마이그레이션 완비 → **도시는 저장하지 않는다. `currentCity` 필드 불요 = 구 §9.8이 지목한 "최대 리스크" 대변경 회피.** 이 하나의 매핑에서 세 난제가 동시에 녹는다: (1) 세이브 대변경 소멸(파생), (2) 코지 이월 자동(layouts·deco·renovated·cleanBy 전부 셸터 키), (3) 숙련 이월 자동(regionVisits 지역 키).

#### 9.8.5 이주 — 대형 프로젝트, 왕복 가능·비싸게

경로 개척 프로젝트 신규 = **0**(예인선·케이블카·선로 복구 완공 = 경로 개통). `moveCostFor`에 cross-city 티어 1겹 + 첫 도착 1회 교두보 할증(`citiesReached`). 왕복 상시(방향마다 재지불).
**계절 창은 pull 수정자, push 아님(코지 안전선 사수)**: 항구=여름·관측=겨울 유리는 **인-시즌 수율/접근 이점**. 예인선 결빙은 깊은 겨울의 **신규 크로스-시티 이동만** 지연, 플레이어를 결코 좌초 안 시킴(어느 도시든 월동 셸터 ≥1 + 완결 생존 루프). "겨울 오기 전 떠나라"(push)로 안 미끄러지게, 창은 **최적 이동 타이밍만 게이트, 생존은 절대 게이트하지 않는다.**

#### 9.8.6 숙련·코지 축적 이월 — 핵심 난제 해법 (3층)

- **① 숙련 이월 = 코드 0.** `regionVisits` 지역 키·8지역 서로소 → 이주=백지 지도, 옛 도시 숙련은 세이브에 얼어붙어 보존(왕복 귀향 시 만렙 그대로). **`revisitDecay`(감쇠) 비채택 — 감쇠=징벌=코지 적대.** 떠남의 무게 = 파괴가 아니라 "두고 온 것이 저기 기다림."
- **② 커스터마이즈 이월 = 코드 0.** `layouts·deco·renovated·cleanBy` 셸터 키·셸터는 도시별 유일 → 이주·왕복해도 삭제 0. **정든 집은 사라지지 않고 기다린다.**
- **③ 정든 정도(stayDays) 이월 = `homeStay` 신규(난제의 진짜 해법).** `state.stayDays`는 스칼라라 이주 시 0 리셋(2336)이고 `endingLeaning` rest 항이 이걸 읽는다(6290) — layouts만 이월되고 "정든 체감"은 리셋. **해법**: `state.homeStay={shelterId:max}` 고수위 기록. 이주 시 stayDays 리셋(온화한 무게=라이브 정착 스트릭 상실) 하되 **귀향 시 `homeStay[id]`에서 복원**("오래 살던 집" — 낯선 집 취급 방지) + `endingLeaning` rest 항은 `max(stayDays, homeStay[current])`를 읽는다. 세이브 1필드 + 읽기 2곳 = 소형.

**결론**: 이주는 리셋이 아니다. 리셋되는 건 라이브 정착 스트릭·현장 숙련뿐, 지은 집·모은 것(inventory·blueprints·paints·knowledge·복장·수집·scars·gun·hazmat)·정든 기억은 전부 보존.

#### 9.8.7 세이브 스키마 + 마이그레이션

`currentCity` 불요(파생). 신규 = 전부 순수 가산 소형(front·scars·gun 패턴):
```js
if (rawState.cityWinters == null) state.cityWinters = {};       // 엔딩 가중 누산기
if (rawState.finalWinterCity === undefined) state.finalWinterCity = null; // 9겨울 체류지
if (rawState.homeStay == null) state.homeStay = {};             // 정든 정도 고수위(9.8.6-③)

```
- 지역 폴드: `regionVisits`의 commercial/industrial/lab 키는 고아 → masteryTier가 현 REGIONS만 읽어 **무해**(선택적 폴드 보존).
- 구세이브: `cityWinters.home = state.winters`(과거 겨울=첫 도시).
- ⚠️ **`cityWinters`는 day 파생 아님** — 체류 이력 = 플레이어 선택 누산기(RNG 0 결정론이되 저장 필드).

#### 9.8.8 엔딩 체류 가중 (§9.5 예약분 실현)

`endingLeaning()`(6286)은 `lean.x +=` 객체가 아니라 **로컬 스칼라 3개 argmax + escape 우선 동률**. 실구조에 얹는다:
```js
const escape = 기존 + (cw.port||0)*W + (fin==='port'?Wf:0);
const newworld = 기존 + (cw.capital||0)*W + (fin==='capital'?Wf:0);
const rest = Math.min(4, Math.floor(Math.max(state.stayDays||0, homeStay[current]||0)/8)) + 기존
           + ((cw.home||0)+(cw.observatory||0))*W + ((fin==='home'||fin==='observatory')?Wf:0);
return escape >= newworld && escape >= rest ? 'escape' : newworld >= rest ? 'newworld' : 'rest'; // ★기존 동률 순서 불변
```
- 도심→신세계 / 관측·첫 도시→안식 / 항구→탈출. **기존 escape-우선 동률 유지**(구세이브 엔딩 조용히 바꾸는 변경 거부).
- **가중 무력화 방지**: 첫 도시 강제 2겨울로 rest 편향되는 걸 **9겨울 체류지 보너스(Wf ≫ W)**로 해소 — "어디서 끝났나"가 완주형에게도 결정력.
- `passWinter`(winters≥9 밤)에 `finalWinterCity=cityOf(current)` + `cityWinters[cityOf(current)]++`. day/winters 파생·RNG 0·sim 무접점. 3택+보류 항상 개방(§9.5 계약 불변).

#### 9.8.9 도시 시그니처 실체 — 복장 렌더는 별도 실측 트랙

**아바타 복장 = 은닉 신규 렌더 시스템임을 정직하게 계상.** 아바타는 복셀 렌더(#86 완료)지만 복장 스왑은 #119가 발판일 뿐 미구현 — catCoat(4종 복셀)과 달리 **실질 신규 렌더 작업**. 스코핑: 복셀 리컬러/부착(에셋 0)이라 착지 가능하되 **별도 트랙 + 렌더 스파이크(실측 게이트)**로 얹는다. `state.wardrobe`(소유)·`state.outfitWorn`(착용) + 옷장 UI. 획득 = 첫 도착 + 도면 + 제작 1회(숙련 티어 아님).

| 도시 | 시그니처 축 | 실체 | 획득 |
|---|---|---|---|
| 첫 도시 | 가구·도료 + 복장 | 기존 시그니처 8종·도료 + 「잿빛 홈스펀」 | 기존 + 첫 도시 도면 |
