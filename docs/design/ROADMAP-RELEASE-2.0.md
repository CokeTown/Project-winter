## B. 2.0 「응답」 밑그림 (콘텐츠 비축 트랙)

### B-1. 무대 — 동부 영토 (§6 확정 모델)
- **세관 관문**(customs)이 동부 진입 게이트 → **부산형 항구도시**. 셸터 4종 기초 모델링 완료(#147 진행):
  펜트하우스(종점·첨탑들의 왕좌) · 다리 관리소(은하수·큰 달 밤하늘 확장 첫 사용) · 세관(검문 캐노피·3년차 식생) · 역 대합실(신광 아트리움).
- 신규 지역 8(2+2+2+2), 도심 시그니처=복장(가구 아님 — 옷장 확장).
- 아트 디렉션: TLOU 감성 + 붉은 노을 팔레트(§금문교 비네트 #146이 톤 기준점).

### B-2. 시스템 — 이미 착지한 것 위에 (§9 완결분 활용)
- 낙진 시계(#122)가 동부 개방 타이밍 게이트로 확장 — "겨울 셋을 넘기면 동쪽 길이 열린다".
- 지역 숙련·대한파 프론트·부상 서사(#123) → 동부 지역에도 동일 문법 적용(신규 코드 최소).
- 적대 다이얼·총(#124)은 동부에서 본격화(서부=저강도 유지 — 코지 코어 보호).
- 엔딩 3분기(#125)·침묵 루트(#126~131)와 동부 서사 접점: 무전 기지 송출 응답자들이 동부에 있다 — **"응답"의 실체**.

### B-3. 실행 순서 (출시 후 패치 사이클)
1. 2.0-α: 세관 관문 개방 + 항구도시 지역 2 + 셸터 1(세관) — 낙진 시계 연동.
2. 2.0-β: 역 대합실 + 다리 관리소 + 지역 4 + 시그니처 복장 1차.
3. 2.0-정식: 펜트하우스(종점) + 지역 8 완성 + 동부 서사(응답) + 엔딩 확장.
- 각 단계 = 독립 패치 발행 가능(비축분을 쪼개 라이브 운영 리듬 확보).
