### 1.0 — 완성 판정 (Phase F 마감)
- **콘텐츠 볼륨**: 인카운터 20+, 메모 30+, 방송 12+, 가구 30+(꾸미기 #13 벽지/바닥재 포함), 셸터 9(전부 외형 개성화 #18), 업적 16+
- **밸런스**: 노말 Day60 시뮬 — 굶는 날 0·자원 40~120 밴드 유지·한파 사망 0 / 하드 — 첫 겨울 실패율(탈진일 ≥3) 20~40% / zen — 완전 방치 30일 생존
- **서사 아크**: 온보딩 7퀘 → 계절 압박 → 9겨울(박사 첫 무전) → 라디오/메모 수집으로 진실 윤곽 → Day10000 엔딩까지 단절 없음. "먼 불빛"과 1.4 복선(관측 위성 언급 방송 1종) 심어둠
- **기술**: 크래시 0(전역 핸들러 리포트 기준), 세이브 왕복 무손실, 3플랫폼(웹/PC/Android) 동시 릴리즈, NSIS 업데이트 원클릭
- **문체**: 전 문안 캐논(1인칭·흐르는 타인) + im-not-strange-ai 룰북 통과
