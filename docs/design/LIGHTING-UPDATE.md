# Lighting Update — 설계 스펙 (Phase 1 착수: 2026-07-16 디렉터 "시작해")

> **출처**: 디렉터 구술 (2026-07-15). §6 단계 분할 중 **1단계 완료** — 2·3·4단계는 후속.
> **선행 착지**: 이미 반영된 라이팅 하향(커밋 `73dd597`, ceilLight 29→11 + 화기 주광원화 + 천장 점멸)이
>   이 업데이트의 1차 방향이자, §1 "폴백 천장광 밝기 10"의 전신이다.
> **정리**: CTO(Fable). 디렉터 원문 의도를 최대한 보존.

---

## 0. 한 줄 요약
빛을 **거저 주어지는 것**에서 **관리·수집·표현의 대상**으로 승격한다. 어둠은 압력(생존), 빛은 보상(코지·꾸미기)이 된다.
"집을 꾸민다 = 조명을 다룬다"를 게임 메카닉으로 만든다. 광원인 만큼 **연출은 '감동' 수준**(빛 번짐·반사)이어야 한다.

---

## 1. 광원을 Tough 하게 (생존 압력)
- **기본 상태 = 어둠**: 양초/화기를 두지 않는 한, **햇빛 외 실내 광원 없음**. "우울할 정도로" 어둡게.
- **폴백 천장광**: 집 안에 광원이 하나도 없으면 — 천장에 형광등이 있는 듯하되 **밝기 10/100 수준**(아주 흐릿, 냉기).
  현재 ceilLight(11)를 이 "무광원 폴백"으로 재정의: **광원 오브젝트가 있으면 폴백 천장광 소등/최소화**, 없으면 밝기 10.
- 의도: 첫 세션의 어둠이 "빛을 마련하라"는 자연 동기 → 양초/화로/조명으로 온기를 '쟁취'하는 루프.

## 2. 시설 업그레이드: 조명 + 전력
- **조명 시설**(셸터 개조/업그레이드 항목): 설치 시 실내를 밝힌다. 단, **전력을 지속 소모**.
- **전력 경제**: 조명·가전이 전력을 계속 먹는다 → 관리 압력(배터리 잔량).
- **태양광(태양열) 설치**: 전력 걱정 해소 + **배터리 추가 생성**. (기존 solar 패널 확장 — 지속 급전으로 승격)
- 밸런스 축: 어둠(무비용·우울) ↔ 화기(연료·온기·흔들림) ↔ 전기조명(전력·안정·꾸미기) ↔ 태양광(초기투자→무한).

## 3. 조명 색 파밍 (코지·꾸미기 세로축)
- **전설급 고밸류 아이템: 조명 색 변경 도구** — 조명의 색을 바꾼다.
  - 근거: cozy라 따뜻한 불빛이 기본값으로 좋지만, **집 꾸미기의 본질은 조명**이다. 색이 곧 분위기.
  - 도료 시스템(「폐허의 팔레트」)의 **조명판**: 색을 파밍/수집해 조명에 입힌다.
- **초희귀 아이템: LED 조명** — 양초·화로 계열 밖의 신규 광원. **색 파밍 가능**.
  - 화기(따뜻·흔들림) 대비 LED(선명·안정·컬러) → 표현 스펙트럼 확장.

## 4. 연출 수준 — "감동" (Ray Tracing은 아니되)
- 광원 업데이트인 만큼 **시각적 감동**이 목표. 풀 레이트레이싱은 아님.
- **필수 표현**: 빛 번짐(bloom/glow), **반사**(바닥·젖은 표면·금속 하이라이트), 부드러운 감쇠, 색 섞임.
- 화기: 따뜻한 색 + 흔들림(flicker, 이미 구현). LED: 선명·안정 + 컬러.
- 픽셀 아트 정합 유지(무디드 팔레트·셀 명암) — 회화적 블룸 남발 금지(캡슐 리포트 §4 AI 느낌 회피와 정합).

## 5. 구현 메모 (착수 시)
- **광원 레지스트리**: "실내 광원 존재 여부"를 집계해 폴백 천장광을 on/off (현 ceilBaseInt 로직 확장).
- **전력 틱**: 조명/가전 전력 소모를 일일 정산(processDay)·실시간 게이지에 연결. 태양광=일일 배터리 급전.
- **색 시스템**: 도료(PAINT_*) 색 팔레트를 조명 color에 매핑. 조명별 `tintColor` 저장(세이브).
- **LED 광원 def**: furniture.js에 신규(색 가변, flicker 없음/미세), 초희귀 드랍 게이트.
- **렌더**: bloom(포스트) + 바닥 반사(기존 젖은 도시 스트릭 재사용) 강화. reduceMotion·lowSpec 경로 존중.
- **밸런스 게이트**: 조명 시설/솔라/색도구/LED의 해금·비용 곡선은 REWARD-LOOP·난이도별 압력과 정합.

     - ceilLight → 무광원 폴백 재정의(냉백 0xcfe0e8·밝기 10·점멸, 광원 있으면 소등). 광원 레지스트리 = 가구 광원(화기 포함)+조명 설비.
     - SHELTER_MODS.lighting(부품3+배터리1, rebuild) — 천장 펜던트 소품+전등(0xffd9a0·16·무점멸), 배터리 1/일(발전기 가동 시 무료), 단전 시 자동 소등+아침 노트.

  3. **조명 색 파밍(도료 조명판)** →
