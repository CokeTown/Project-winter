# Nine Winters — 확장 기획서 1.1~1.4 "아홉 번의 겨울, 네 개의 지평선"

> 원칙: ①맵이 아니라 "셸터에서 닿는 세계의 결"을 확장 ②각 확장은 기존 시스템의 미완 고리를 하나씩 회수 ③캐논 준수(정착자는 나뿐, 타인은 흐른다, 전 문구 1인칭) ④모든 재난은 예보→대비→서사 3박자.
> 공통 뼈대 **대형 프로젝트 시스템**은 1.1에서 구현, 이후 재사용.

---

## 공통 시스템: 대형 프로젝트 (1.1 신규, 전 확장 재사용)

여러 날에 걸쳐 자재를 붓는 장기 목표. 후반 자원 싱크이자 "다음 겨울의 목표".

- `state.projects[id] = { stage, invested: {res: n} }` — 단계별 요구 자재 테이블(BAL.projects)
- UI: 프로젝트 현장(3D 오브젝트가 단계별로 변함 — 뼈대→반쯤→완성) 클릭 → 투입 모달(자원 칩 보유/필요 대조 — 이주 칩 문법 재사용)
- 하루 1회 "작업" 행동(에너지 15): 투입된 자재를 진행도로 변환 — 방치와 능동 사이의 완급
- 완성 보상은 **기능**(이동로/생산/방어)이지 숫자 보상 아님. 수첩에 "그 해의 공사" 자동 기록(memoir 문법)
- 게이트: 프로젝트 1개당 총 자재량 = 노말 기준 12~20일치 잉여분(시뮬로 캘리브레이션)

## 1.1 「얼어붙은 항구」 — 바다의 결 (회수: 여름 부패, 겨울의 보상 부재, 개 동료)

**픽션 한 줄**: 강 하구를 따라 내려가면, 얼어붙은 항구가 있다. 바다는 죽지 않았다.

### 신규 지역 2
| 지역 | 성격 | 특화 |
|---|---|---|
| 항만 야적장 | 중위험 · "오늘 바다가 준 것" — 컨테이너 화물 테이블이 매일 리롤(복권 파밍) | 랜덤 편중 드랍(어느 날은 천 12, 어느 날은 부품 6) |
| 수산시장 폐허 | 저위험 | 소금 주 산지 + 신선식품, 겨울엔 드랍 절반(결빙) |

### 신규 시스템
- **소금 + 염장**: 제작대 "염장 선반"(가구) — 신선식품 2+소금 1 → 보존식 2. 냉장고 없는 초반의 대체 해법, 여름 부패 압박의 카운터. BAL.economy.saltCure
- **얼음낚시(겨울 전용)**: 겨울에만 항구 빙판 스팟 개방 — 낚시 행동(에너지 10, 2게임시간): 신선식품 1~3 + 극저확률 "이상한 병"(메모 병 편지 3종). **겨울이 처음으로 '받는 계절'이 된다** — 계절 압박의 반대급부
- **개 동료** (1.0에서 이동): 항구 인카운터로 1회성 등장(고양이 문법) — 유지비 음식↑, 침입 이벤트 자동 방어 1회/겨울, 탐험 성공률 +3%p. 고양이=위안, 개=일. 캐논: 동물은 정착의 예외
- **대형 프로젝트: 방파제 오두막** — 항구에 전초기지(파밍 시간 -25%, 얼음낚시 스팟 +1)

### 인카운터 6+ / 밀수꾼 행상인(계절 가격 극단 — 겨울 연료 3배) / 메모 8종(항구: 마지막 배, 못 떠난 사람들)
### 에셋: 지역 아이콘 2, 셸터(예인선·관제탑) 2+아이콘, 소금/염장/낚시 아이콘, 항구 이벤트 일러스트 2, 종이 지도 확장판(map_paper_v2 — 하단에 항구 추가)
### 게이트: 여름 시뮬에서 염장 채택 시 부패 손실 60%↓ / 겨울 식량 곡선이 낚시로 완만 / 개+고양이 동시 생존 30일

## 1.2 「지하 노선도」 — 발밑의 결 (회수: 후반 인플레, 지역 이동 시간, 지하철 셸터 존재감)

**픽션 한 줄**: 도시의 혈관은 아직 뚫려 있다. 어둠 속에서만.

### 핵심: 선로 복구 (대형 프로젝트 × 3구간)
- 지하철 셸터가 허브로 승격. 구간별 복구(잔해 제거→침목→개통): 개통 구간의 지역은 **탐험 시간 50%↓ + 겨울 폭설 봉쇄 무시**(지하니까) — 계절 압박의 우회로를 "건설로 산다"
- 수동 궤도차(핸드카) 연출 — 출발 SFX/짧은 암전 프레임

### 신규 시스템
- **버섯 재배칸**(가구): 빛 없이 식량 — 물 소모, 지하/실내 셸터 보너스. 온실의 대칭축
- **암시장**(개통 후 개방): 잉여 자원 물물교환 — 교환비는 계절 변동(겨울 연료 프리미엄). **후반 인플레의 최종 싱크**. 화폐 없음(캐논: 문명은 죽었다, 교환만 남았다)
- 지하 전용 인카운터: 터널의 소리(정체 불명·서사), 궤도 위의 유모차(메모), 버섯 군락 발견

### 메모 10종: 판데믹 초기 지하 대피 서사의 본진(격리 안내 방송문, 셔터 앞의 줄, 마지막 열차)
### 에셋: 암시장/버섯/핸드카 아이콘, 터널 이벤트 일러스트 2, 지하 노선도(수첩 탭 전용 손그림 노선도 1)
### 게이트: 개통 전후 탐험 시간 실측 -50% / Day90 시뮬 자원 상한 밴드 유지(암시장 싱크 작동) / 폭설 중 개통 구간 탐험 가능

## 1.3 「고요한 고원」 — 높이의 결 (회수: 겨울 콘텐츠 심화, 쾌적함의 정점, 감상 보상)

**픽션 한 줄**: 높은 곳의 겨울은 더 혹독하고, 더 아름답다.

### 신규 지역: 리조트 폐허(고위험·고보상 — 호텔 물자) / 신규 셸터: 스키 로지(최고 단열 기본, 벽난로 붙박이, viewH 큼 — 전망)
- **고도 페널티**: 고원 셸터 거주 시 연료 소모 +30%, 한파 빈도 +1 — 로지의 단열이 상쇄(리스크·리워드 셸터)
- **눈사태**: 겨울 재난 2호 — 예보(우르릉)→ 고원 지역 3일 봉쇄 or "위험 우회로"(성공률 -15%p, 보상 1.5배) 선택지. 원칙 준수: 예보 있고, 선택이 있다
- **온천**(로지 전용 개조): 에너지 회복 +대형, 쾌적 온기 축 +10 — cozy의 정점. 고양이/개가 온천 옆에서 조는 전용 포즈
- **관측소 (대형 프로젝트)**: 완성 시 밤하늘 이벤트 개방 — 유성우/오로라/인공위성 궤적(1.4 복선: "저건 별이 아니다"). 감상 보상 = 수첩 스케치 수집 6종(makePaperTexture 스케치 문법)
- **케이블카 복구** (대형 프로젝트): 고원↔도심 연결 — 리조트 접근 시간 단축

### 메모 8종(리조트: 마지막 휴가객들) / 에셋: 로지·온천·관측소 오브젝트, 밤하늘 스케치 6, 눈사태 예보 아이콘
### 게이트: 로지 겨울 시뮬 — 단열 미비 시 연료 파산, 완비 시 최고 쾌적(리스크·리워드 실측) / 눈사태 봉쇄 중 우회 선택 동작

## 1.4 「금지 구역」 — 진실의 결 (회수: 세계관 전체, 라디오 수집, Day10000 다리)

**픽션 한 줄**: 군이 마지막으로 지킨 곳. 그들이 무엇으로부터 도망쳤는지, 거기 적혀 있다.

### 진입 게이트: **방호복**(고급 제작 정점 — 희귀부품 다수+천+절단기 계보) 없이는 입장 불가. 체류 시간제(방호복 내구 — 탐험 1회당 소모, 수리 가능)
### 진입로 2갈래 (2026-07-05 디렉터 확정): ①격리 검문소 정공법 ②**돔 벙커 지하 계단**(v0.9.6~7에 전실+하강 계단 복선 설치 — #55) — 벙커=군용 시설이 군 지하망으로 이어진다. 벙커 거주자의 보상 서사: "구멍 난 피난처 → 집 → 아래로 열리는 문"
### 신규 지역: 격리 검문소(중위험) → 지하 연구동(고위험·방호복 필수) — 2단 진입 구조
- 드랍: 희귀부품 최다 + **기밀 문서 메모 12종** — 판데믹의 기원, 폭격 결정의 기록, 그리고 "관측 프로그램"의 존재. 수집률 100% 시 수첩에 최종장 "그날의 진실" 자동 페이지
- **무전 기지 복구 (최종 대형 프로젝트)**: 완성 → 수집한 라디오 방송을 **송출**하는 전용 행동. 송출할 때마다 세계 어딘가의 응답 — 종이 지도에 작은 불빛이 하나씩 켜진다(맵 오버레이). 모든 방송 송출 시: 박사의 정기 교신 개방 → Day10000 엔딩으로 가는 서사 다리 완성. **게임 주제의 회수: "폐허에도 온기는 남아 있다" — 내가 지킨 온기가 신호가 되어 퍼진다**
- 인카운터: 자동 방송 루프(기계 목소리), 방호복의 낯선 발자국(끝내 만나지 않는다 — 캐논), 새들이 돌아온 자리

### 엔딩 정책: 1.4 이후에도 게임은 끝나지 않는다(zen 정신) — 엔딩은 "볼 수 있는 것"이지 "끝내는 것"이 아님
### 에셋: 방호복 아이콘/착용 표시, 검문소·연구동 지역 아이콘, 기밀 문서 전용 종이 텍스처(타자기 문서), 무전 기지 오브젝트 3단계, 지도 불빛 오버레이
### 게이트: 방호복 없이 진입 차단 실측 / 문서 12종 수집→최종장 발화 / 송출→불빛 점등 왕복 저장 무손실 / 신규 유저가 1.0 콘텐츠만으로 방호복 계보 도달 가능(진입 장벽 검증)

## 릴리즈 케이던스 제안
