# Nine Winters 현지화 가이드 (TR-2)

> 근거: 게임메카 일본인 게임 번역가 인터뷰(2026-07-09, 디렉터 공유) — "번역가에게 문맥을 주는 시트가 품질을 결정한다."
> 짝 문서: `docs/l10n/translation-sheet.tsv` (생성: `node tools/export-l10n-sheet.mjs`)

## 1. 시트 규격
- 탭 구분(TSV) · 제목 행 1개 · 셀 병합 없음 (CAT 도구/AI 어느 쪽이든 그대로 투입 가능)
- 컬럼: `id · context(출력 위치/용도) · tone(어투 권고) · vars(치환 변수) · chars_ko(원문 글자수) · ko · en · ja · zh_cn`
- `{n}` `{name}` 같은 변수는 **원형 그대로 유지** — 게임이 런타임 치환한다.
- `\n`은 개행 이스케이프 — 번역문에서도 동일 표기 유지.

## 2. 어투(tone) 매핑 — #139 어투 대개편이 번역의 헌법
| tone | 한국어 원칙 | 일본어(ja) | 중국어 간체(zh_cn) |
|---|---|---|---|
| 혼잣말·반말 | 플레이어 행동·사건 = 생존자의 독백 | 独白体 — だ・である/タメ口, 一人称は「私」 생략 위주 | 구어체 독백, 짧은 문장 |
| 시스템·다나까 | 설정/확인창 = 격식 존대 | です・ます체 | 표준 서면어(请/将) |
| 문어체(기록) | 메모·유서·회고 = 글로 남긴 것 | 書き言葉, 체언 종결 허용 | 서면어, 담담한 톤 |
| 라벨(명사구) | 버튼·HUD = 명사구, 잘림 주의 | 명사구, 글자수 ko 대비 ±20% | 명사구, 축약 우선 |

- **지시조 금지**는 전 언어 공통(세계관 정합 — "정착자는 나뿐").
- 대사 끊김(…)·말줄임은 원문 리듬 유지. 이모지 프리픽스(🔥 등)는 번역하지 않고 유지.

## 4. 워크플로
1. `node tools/export-l10n-sheet.mjs` → 최신 시트 재생성 (키 추가 시마다).
2. ja/zh 초벌: AI 번역에 **context·tone 컬럼을 함께 투입** (이 문서 §2·§3 첨부).
3. LQA: 원어민/커뮤니티 검수 — 시트의 빈 칸이 아니라 **게임 화면에서** 확인(잘림·조사).
4. 반영: 시트 → `src/locales/<lang>.json` 변환(도구 추후) → `check-i18n` 확장.
5. Steam AI 공개 문구에 번역 AI 사용 명시(기존 문구에 포함됨 — STORE-SUBMIT.md).

## 5. 남은 것
- ✅ 데이터 테이블 병기 문자열(#114 Phase 2, 2026-07-15): `data.*` 542키로 로케일 JSON 합류.
  - 파이프라인: 데이터 표의 *En 병기 필드 → `node tools/export-data-i18n.mjs` → `src/locales/*.json`(+public cp)
  - 근원 = `src/data/l10n-registry.js`(표 등록·자동 발견). 신규 *En 필드는 제너레이터 재실행으로 합류.
  - 런타임: 부팅 시 `stampDataL10n()`이 비열거 `_lk`를 스탬프 → i18n `LF/LC`가 JSON 우선, 병기 필드는 폴백.
  - 배열 병기(colorNames)는 파이프(`|`) 결합 단일 키 — 번역 시 파이프 개수 유지.
  - 스코프 밖: game.js 잔류 표(WEATHERS·ACHS — #73 분리 후 합류) · decotex.js(런타임 클로저 생성).
- 인카운터·대사 전량(#141 추출본)은 문맥 컬럼이 특히 중요 — 화자/상황 주석을 그 파일에서 가져온다.
- 우선순위: 데모(넥스트 페스트) → ja → zh_cn (DPM 사례: 일/중/한 페이지 번역 권장).
