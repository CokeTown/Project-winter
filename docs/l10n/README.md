# 번역 추출 패키지 — Gemini 번역 워크플로용 (#191)

> 디렉터가 Gemini 등 외부 LLM으로 UI·게임플레이 번역을 직접 돌리기 위한 추출본.
> 게임에 반영되는 정본은 `src/locales/*.json`이며, 이 폴더는 **스냅샷**이다.

## 갱신 (자동 아님 — 이 명령을 돌려야 최신이 된다)

```powershell
node tools/export-l10n-pack.mjs
```

`full` · `chunks` · `delta` · `translation-sheet.tsv` · `CHANGES.md`가 한 번에 다시 쓰인다.
게임에서 언어를 바꾸는 것과는 무관하고, 문자열이 늘거나 원문이 바뀌어도 자동으로 따라오지 않는다 —
그래서 07-17 추출본이 2,068키에 멈춘 채 현행 2,227키와 어긋나 있었다(생성기가 저장소에 없었던 탓).

## 구성

| 경로 | 내용 |
|---|---|
| `CHANGES.md` | **먼저 볼 것** — 신규/변경/삭제 키 수와 목록 |
| `delta/delta-*.json` | **작업 대상** — 기준선 이후 새로 생기거나 원문이 바뀐 키만 |
| `full/{ko,en,ja}.json` | 현행 전체 (ko가 번역의 기준) |
| `chunks/chunk-*.json` | 전량 청크 85키 단위 — 통번역을 다시 할 때만 |
| `translation-sheet.tsv` | 시트/CAT 도구용 탭 구분 표 |

### 기준선(델타의 시작점)

`.baseline-ko.json`이 "마지막으로 번역을 반영한 시점의 원문"이다. 재추출을 몇 번 하든 델타는 그대로 남고,
번역을 실제로 역수입한 뒤에만 기준선을 옮긴다:

```powershell
node tools/export-l10n-pack.mjs --set-baseline
```

## Gemini에 넣을 때 지켜야 할 규칙 (프롬프트에 포함 권장)

1. **플레이스홀더 보존**: `{n}`, `{item}` 같은 `{...}`는 글자 그대로 남긴다. 개수·이름이 바뀌면 게이트가 거부한다.
2. **{josa}는 한국어 전용**: 영어/일본어 결과에서는 제거하고 자연스러운 문법으로 풀어쓴다.
3. **HTML 보존**: `<br>`, `<b>` 등 태그는 구조 그대로.
4. **톤 규약**: 플레이어 행동·결과=혼잣말 반말(일본어 常体 독백), 시스템 안내=존댓말(です・ます), 버튼·라벨=간결형. 일본어 상세 규약·용어집은 `docs/design/L10N-JA.md`.
5. **출력 형태**: `{ "키": "번역문" }` 평면 JSON — 이 형태 그대로 역수입 도구에 넣는다.

프롬프트 예시:
```
아래 JSON의 각 항목을 일본어로 번역해줘. ko가 원문, en은 참고.
{...}(플레이스홀더)와 <br> 태그는 그대로 보존. {josa}는 삭제하고 조사를 직접 써.
행동/결과 문장은 독백체(常体), 시스템 안내는 です・ます, 버튼 라벨은 간결한 체언 종결.
출력은 { "키": "번역문" } 형태의 JSON만.
[chunk-01.json 내용 붙여넣기]
```

## 번역 결과 게임에 반영하기 (역수입)

```powershell
node tools/import-l10n.mjs ja 결과1.json 결과2.json   # 또는 en
node tools/check-i18n.mjs                              # 3로케일 무결 게이트
npm run test:build                                     # 전체 배터리
```

- 역수입 도구가 **미지 키·플레이스홀더 불일치·한글 잔존·{josa} 잔존**을 전부 검사하고,
  위반이 하나라도 있으면 **아무것도 적용하지 않는다**(부분 오염 방지). 위반 목록을 보고 해당 키만 고쳐 재시도.
- 한국어 원문(`src/locales/ko.json`)을 직접 고쳐도 된다 — 고친 뒤 en/ja도 함께 갱신하고 게이트를 돌리면 끝.

## 재추출 (원문이 바뀐 뒤 이 폴더 갱신)

세션에서 "l10n 추출 패키지 갱신해줘"라고 하면 full/chunks를 현행 기준으로 다시 뽑는다.
