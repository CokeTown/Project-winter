## 1. 🔴 계절을 "마감일"로 (— #15 계절 압박)

지금 계절은 배경이다. 코지 생존의 긴장은 "겨울이 온다"에서 나와야 한다.
- 가을 = 비축 시즌: "겨울까지 N일 — 연료 X, 보존식 Y 권장" 경고 카드 (3일 전부터)
- 겨울 = 보스: 배고픔 가속(있음) + 한파 이벤트(신규: 2~3일 예보→단열/연료로 방어→결산에 방어 성공 서사)
- 여름 = 부패/갈증 압박(통조림의 존재 이유 강화), 봄 = 파종/청결
- 수치는 balance 테이블로 분리 (하드코딩 제거)

**이게 1순위인 이유**: 중반(Day 20~50) 루틴화를 깨는 가장 저렴하고 확실한 장치. 기존 시스템(예보/개조/자원)을 전부 재사용한다.

## 2. 🔴 장기 목표: "아홉 번째 겨울" (— #11)

제목이 곧 약속이 되게 한다.
- HUD/타이틀에 겨울 카운터 (❄️ 2/9)
- 겨울을 날 때마다 기념 비트: 수첩에 자동 기록되는 "그 해 겨울" 페이지(그 계절의 사건 요약) — 셸터 앨범의 씨앗
- 9번째 겨울(≈Day 400) = 대형 마일스톤: 특별 연출 + 업적 + (엔딩 힌트: 박사의 첫 무전)
- Day10000 엔딩은 유지하되, 9겨울 이후 "관측 위성이 당신을 발견했다" 같은 중간 서사 비트로 다리를 놓는다

## 3. 🟡 서사 공급망: 라디오 방송 수집 (— #12)

인카운터 8종은 반복이 빨리 체감된다. 확장의 축은 "양"이 아니라 **상태 조건부 + 서사 조각**:
- 라디오 방송 10~15종 (날씨예보/행상인 예고/위험 경고/과거 정부 방송/정체불명 음악/생존자 사연/박사의 실험 일지 조각)
- 한 번 들은 방송은 수첩 라디오 로그에 수집 (수집률 표시) — 조각을 모으면 세계의 진실이 맞춰진다
- 떠돌이 개 동료화(음식 유지비, 침입 방어) — 고양이와 다른 역할군
- 행상인 계절 가격 변동 (겨울 전 연료 비쌈 = 가을 비축 동기와 맞물림)

## 4. 🟡 경제: 후반 소비처 (— #15, #29)

Day 40+ 자원이 쌓이기 시작한다. 싱크(소비처) 후보:
- 고급 제작 트리: 희귀부품 기반 상위 가전(자동 급수기/온풍기) — 희귀부품에 "장기 사용처" 부여
- 거처 개조 2단계(강화 단열재/대형 빗물받이) — 비용 곡선 상향
- 상인에게 잉여 자원 판매(물물교환) → 꾸미기 전용 재화? (과설계 주의 — 베타 데이터 보고 결정)

## 5. 🟡 긴장 원천 다양화

- 계절 재난(§1의 한파/폭염) 외: 3일 폭설(탐험 봉쇄 → 비축 시험), 짙은 안개(성공률 감소 but 희귀 루트 확률↑ — 리스크 선택)
- 원칙 유지: 실패 = 사망이 아니라 생활비 증가. 모든 재난은 예보+대비 수단이 반드시 존재
