# Nine Winters — 트레일러/스크린샷 연출 브리프 (디렉터 방향, 2026-07-05)

> 디렉터 원문: "집 꾸미기와 아늑한 분위기, 탐험하고 확장되는 세계. 회색빛이지만 아름답고, 그만큼 절망적인 세상을 그리고 싶다."
> 이 문서가 트레일러 편집·스크린샷 선별·스팀 캡슐 무드의 단일 기준이다. GD-THESIS와 한 몸.

## 무드 공식
**회색 세계 × 앰버 온기의 병치.** 절망과 아름다움은 같은 프레임 안에 있어야 한다 — 잿빛 폐허 와이드샷의 구석에 켜진 창문 하나. 절망을 따로 찍고 아늑함을 따로 찍으면 실패. 모든 컷의 심사 기준: "이 프레임에 차가움과 따뜻함이 공존하는가?"

## 트레일러 3막 구조 (45~60초, BGM: Main_theme)
**1막 「온기」 (0~18s)** — 실내에서 시작한다. 게임의 약속을 먼저.
- 난롯불 클로즈업 → 고양이 쓰다듬기(눈 감김) → 꾸미기 컷 전환 연속(빈 컨테이너→가구 차오름, 3~4컷 점프) → 창밖으로 눈 내리는 창가 빛기둥
**2막 「바깥」 (18~38s)** — 문을 연다. 세계가 넓어진다.
- 종이 지도 펼침 → 탐험 출발 → 지역 몽타주: 얼어붙은 항구(페리 황혼)/지하 터널(핸드카)/고원 설산(케이블카·오로라) → 계절 전환(봄 새순→겨울 나목, 같은 앵글)
**3막 「회색의 아름다움」 (38~58s)** — 절망과 응답.
- 잿빛 폐허 와이드(도심 헤이즈) → 한파, 창유리 성에 → 금지 구역 실루엣 → **무전 송출 → 종이 지도에 생존자 불빛이 하나씩 켜진다** → 암전 → "폐허에도 온기는 남아 있다" → Nine Winters 로고 → Wishlist Now
- 마지막 30프레임: 난로 앞 고양이(1막 수미상관)

## 스크린샷 8장 셀렉션 기준 (스팀 페이지)
1. 히어로: 눈 오는 황혼, 꾸며진 컨테이너 실내+창밖 폐허 (병치 공식의 대표컷)
2. 꾸미기 만렙 실내(테마 세트+벽지)
3. 고양이 클로즈업(쓰다듬기)
4. 종이 지도(전 지역+생존자 불빛 점등 상태)
5. 페리 or 등대 황혼(확장 세계)
6. 고원 오로라+관측소
7. 한파 방어 장면(성에 낀 창+난로)
8. 겨울 와이드(잿빛 설원 속 셸터 불빛 하나)
