## 1. 1280×800 UI 오디트 — 통과

| 화면 | 판정 | 비고 |
|---|---|---|
| 타이틀 | ✅ | 언어 3버튼 포함 전부 뷰포트 내 |
| 인게임 HUD+패널 | ✅ | uiz 1.25 자동 적용, 겹침·잘림 없음 |
| 수첩(4탭) | ✅ | |
| 제작대 | ✅ | 모달 내부 스크롤(스크롤바 실존) — 폴드 아래 항목은 스크롤 도달 |
| 지도(구역과 거처) | ✅ | 동일 — 모달 하단 닫기 버튼은 스크롤 끝에 있으나 패드 B로 즉시 닫힘 |
| 설정 | ✅ | |
| 자유 텍스트 입력 | ✅ 없음 | range/checkbox/select뿐 — 덱 온스크린 키보드 의존 없음 |

프로브 노트: 자동 오버플로 검사에서 제작/지도 모달이 걸렸으나 **내부 스크롤 컨테이너의 정상 동작**으로 판정(스크린샷 확인). 실제 뷰포트 이탈 요소 0.

## 2. 게임패드 (Standard Gamepad 매핑) — 통과

| 기능 | 판정 | 근거 |
|---|---|---|
| 좌스틱 → 가상 커서 표시·이동 | ✅ | 스텁 패드 주입, padState active+좌표 이동 확인 |
| A → 합성 클릭 | ✅ | pointer/mouse/click 표준 디스패치 — 실핸들러(Y·B) 경유로 메커니즘 검증 |
| B → 닫기 스택 (설정→배치→선택→모달→확인창) | ✅ | 설정 닫힘 어서션 통과 |
| Y → 배치 모드 토글 | ✅ | 툴바 표시 어서션 통과 (타이틀 중엔 무시 — 의도) |
| LB/RB 줌 · Start 일시정지 · 우스틱 회전 | 코드 확인 | 동일 폴링 루프 — 실기에서 체감 확인 항목 |
