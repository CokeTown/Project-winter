# 게임플레이 트레일러 V5 — 컷 시트 (촬영 정본)

> 시놉시스: `STORYBOARD-V5-SYNOPSIS.md` (디렉터 구술). 이 문서는 그것의 **촬영 가능한 형태**.
> 규격: 1920×1080 · 30fps · 총 ~30초. 캡처 `tools/trailer-cap5.cjs` · 조립 `tools/trailer-assemble5.mjs`.
> 자막은 **인게임 DOM 오버레이**로 촬영 시점에 얹는다 — 게임 자체 폰트(둥근모)라 스타일 이질감 0,
> ffmpeg drawtext 폰트 이슈·오탈자 리스크도 없다. 페이드는 프레임 인덱스 기반(결정론).

## 원칙
- **1~3번은 UI 노출이 필수** — "게임이 뭘 하는 것인지"를 3초 안에 보여주는 구간. 나머지는 UI를 숨겨 씬만.
- 셸터 몽타주 전환은 컷 투 컷(크로스디졸브 최소) — 기존 트레일러 전환 감각 계승.
- 자막은 화면 하단 중앙, 클립당 1개, 8f 페이드인 / 12f 페이드아웃.
- 오디오는 조립 단계에서 `music/Main_Theme_Orchestra2.mp3` 믹스.
