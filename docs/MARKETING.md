## 3. Steam — $100 조기 집행 근거와 절차

**결론: 페이지를 먼저 열고, 출시는 나중에.** 위시리스트는 페이지 공개일부터 쌓이고, 출시 주간 알고리즘 노출은 위시리스트 수에 비례한다. $100은 게임 10장 값 — 페이지가 3개월 일찍 열리면 그 몇 배를 회수한다.

절차: Steamworks 가입($100) → 앱 등록 → 스토어 페이지 제출(에셋은 아래 §5, 심사 3~5일) → '출시 예정' 공개 → 데모 업로드 → Next Fest 신청(연 2회, 데모 필수) → 출시.

일정 제안: **지금 페이지 공개 → 데모(Day 1~12 체험판) 1개월 내 → 다음 Next Fest 참가 → 정식 출시는 위시리스트 5,000+ 시점**.

## 4. 채널 플랜 (주간 루틴)

- **X/트위터**: 주 2~3회 15초 클립 (#cozygames #pixelart #indiegame #screenshotsaturday). 고양이 등반, 폭우 창가, 위젯 모드 데모가 3대 소재
- **레딧**: r/CozyGamers(스토리텔링 톤), r/WebGames(웹 데모 링크), r/IndieDev(개발기) — 각 서브레딧 월 1회 이하, 자기홍보 규칙 준수
- **한국**: 트게더 게시판 개설, 스팀 페이지 한국어 완비(이미 i18n 완료가 강점)

## 6. 트레일러 샷리스트 (30초, 무자막으로도 통해야 함)

1. (0–4s) 암전 → 성냥 긋는 소리 → 촛불 켜지며 방이 드러남 (candle_light.ogg 그대로 사용)
2. (4–9s) 창밖 눈 → 서리 결정이 화면에 핌 → Winter OST 페이드인
3. (9–14s) 가구 배치 몽타주 (배치음 리듬에 맞춰 3컷)
4. (14–18s) 폭우 — 젖은 벽 반사, 화면에 빗방울
5. (18–23s) 고양이 등장 → 테이블 폴짝 → 식빵 자세
6. (23–27s) 줌아웃: 모니터 구석 위젯 모드로 게임이 떠 있고 옆에 작업 창 (실사용 화면)
7. (27–30s) 로고 + "Wishlist on Steam"
제작: OBS로 1080p60 녹화 → 컷 편집. 전량 인게임 소스로 제작 가능.
