## 1.0 이후 — 확장 시나리오 "아홉 번의 겨울, 네 개의 지평선" (2026-07-05 확정 방향)
> 원칙: 맵 확장이 아니라 "셸터에서 닿는 세계의 결"을 하나씩. 각 확장 = 바이옴(지역+셸터+자원+이벤트+대형 프로젝트) 한 세트.
- **1.1 얼어붙은 항구**: 항만 야적장(랜덤 화물)·수산시장 / 예인선·관제탑 셸터 / 신규 자원 소금(염장 제작=여름 부패 카운터) / 얼음낚시(겨울이 "받는 계절"이 됨) / 밀수꾼 행상인
- **1.2 지하 노선도**: 지하철 허브 승격 — 선로 복구 프로젝트로 구간 개통(탐험 단축+겨울 봉쇄 무시) / 버섯 재배 / 암시장(잉여 물물교환=후반 싱크 완성) / 판데믹 대피 서사 메모
- **1.3 고요한 고원**: 스키 로지(최고 단열)·케이블카 프로젝트 / 눈사태(예보→우회 선택) / 온천(쾌적 대형 보너스) / 관측소 밤하늘 수집(유성우·오로라 스케치)
- **1.4 금지 구역(엔드게임)**: 방호복 제작 정점 / 세계관의 답(폭격 이유·박사 정체) — #35 종착지 / 무전 기지 복구 → 수집 방송 송출 → 지도의 생존자 불빛 점등 연출("폐허에도 온기는 남아 있다" 회수) / 9겨울↔Day10000 다리
- **공통 뼈대: 대형 프로젝트 시스템**(여러 날 자재 투입 장기 목표) — 1.1에서 구현, 이후 재사용. 후반 싱크+"다음 겨울의 목표" 장치
- 보류: 커뮤니티/정착 시스템 — 고독 정체성 훼손. 하더라도 "불빛이 보이는 이웃" 간접 교류까지만
