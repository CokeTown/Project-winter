## v0.9.5 Beta (2026-07-05) — 무한 모드 + 종이 지도 + 고양이 리워크

**새 콘텐츠**
- ♾️ **무한 모드**: 노말 밸런스 + 넉넉한 시작 물자 + 자동 진행이 처음부터 — 느긋한 방치용. 겨울 카운터도 ❄️ n (약속 없이 세월만)
- 🗺️ **종이 지도**: 탐험 지도가 생존자의 손그림 지도로 — 지구 그림 위 마커를 눌러 출발

**고양이**
- 앉기 모션 수술(판자 실루엣 → 자연스러운 앉음), "배 까고 드러눕기" 추가
- 얼굴이 마인크래프트식 픽셀 텍스처로 (무서운 초록 눈 퇴출)
- 가구를 통과하지 않고 돌아서 다닙니다 + 그림자가 실시간으로 따라옵니다

**보이는 것/편의**
- UI 아이콘 22종 선명화(원색·고대비·프레임 채움) + 아이콘 크기 확대
- 전체 UI 8% 추가 확대 ("안 보인다" 피드백)
- 창가 빛기둥이 진짜 빛처럼 퍼집니다(산란·헤이즈·바닥 웅덩이)
- 가구를 누르면 배치 모드 전환을 물어봅니다(소리+팝업)
- 디스플레이 설정에 ✓ 적용 버튼
- "어디로 가시겠습니까?" 류 설명조 문구 제거

**수정**
- **PC 설치본 업데이트 안 되던 문제**(단일 인스턴스 잠금) — 이번 한 번만 수동으로 기존 앱 완전 종료 후 설치하면, 다음부터는 원클릭 업데이트
- 셸터 주변 차량 공중부양 전수 수정 (9종 순회 확인)

## v0.9.4 Beta (2026-07-05) — 계절 압박 + 아홉 번째 겨울 + 아트 아이콘

**게임의 뼈대**
- ❄️ **아홉 번째 겨울**: 겨울을 넘길 때마다 HUD 카운터(❄️ n/9)가 오르고, 수첩에 "그 해 겨울" 페이지가 자동 기록됩니다. 9번째 겨울 밤, 라디오가 있다면… 무언가 들려옵니다
- **계절이 마감일이 됩니다**: 가을엔 비축 경고(권장 연료/보존식 대조), 겨울엔 **한파**(2일 전 예보 → 단열+난방으로 방어 → 결산 서사), 여름엔 부패 가속
- 새 가전: 온풍기(한파 방어)·자동 급수기 / 개조 2단계: 강화 단열재·대형 빗물받이
- 후반 자원 인플레 해소(Day30 음식 261→50대), 지역마다 가야 할 이유(공업=부품/연료 확정)

**보이는 것들**
- 이모지 UI → **생성 아트 아이콘 63종** (자원/게이지/버튼/가구/셸터/지역/날씨)
- 떠돌이 생존자·침입의 흔적 이벤트에 일러스트
- 진짜 게임식 확인창(브라우저 알림창 퇴출) — "출발한다 / 그만둔다"
- 쾌적함이 4요소(온기/청결/안정감/분위기)로 — 일지에서 "왜 올랐는지" 원인까지
- 탐험 결산에 인과 한 줄: "🔦 손전등으로 어두운 구석까지 뒤질 수 있었다"
- 아침 보고 상단에 "오늘" 브리핑(예보+경고+권장 행동)

**편의**
- Day 10 자동 진행 해금 시 전용 안내 팝업(+지금 켜기)
- 버튼 설명이 게임식 라벨로(호버/퀘스트 유도 시), 하단 배치 창은 🔧 배치 모드 전용
- PC 디스플레이 설정: 화면 모드 3종 + 해상도 프리셋 12종, 위젯 미니창 수정
- 한국어 문장 다듬기 + 어절 단위 줄바꿈(keep-all), 탐험 문구 1인칭 통일
- QA 치트(타이틀 버전 5연타) + 만렙 세이브(docs/qa) — 테스트용, 업적 잠금

## v0.9.3 Beta (2026-07-04) — 안정화 + 난이도 + 개연성 + 편의성 + 라이팅

**치명 수정**
- 터치/클릭 위치가 UI와 어긋나며 조작할수록 밀리던 문제 (좌표 배율 단일화)
- 불러오기 직후 부팅이 검은 화면에서 멈추던 크래시
- 처음 실행인데 "이어하기"가 생기던 유령 세이브 (타이틀 설정 조작이 세이브를 만들던 문제)
- 설정 창이 화면 좌상단(상태바 밑)에 깔려 열리던 근본 원인 수정 (숨김 패널 위치 파괴)

**난이도 모드**
- 새 게임 시 선택: 🌿 노말 / 🔥 하드 (전리품 −30% · 배고픔/갈증 소모 +50%)
- 슬롯 카드와 이어하기에 하드 🔥 뱃지, 기존 세이브는 노말로 인식

**개연성 (게임 흐름)**
- 하루 정산은 이제 "아침에 눈뜨며" 확인 — 자정을 넘기면 화면이 어두워지며 잠들고, 보고는 기상 후
- 침입 흔적 같은 발견형 사건은 기상하거나 탐험에서 돌아왔을 때만 나타남
- 탐험 중 자정을 넘기면 "밤새 폐허를 뒤졌다" (쪽잠 회복 없음)
- 시간 표기를 게임 시간으로 통일: 출발 (30분) · 돌아오기까지 N분 · 부상 약 12시간

**조작/편의**
- 🔧 배치 모드: 평소엔 가구가 실수로 잡히지 않음 — 라디오/조명 켜기 등 기능만 동작, 배치 모드에서만 이동·회수
- 이주 창: 필요 물자를 보유/필요 대조 칩(1/2)으로 표시, 부족분이 한눈에
- 🏠 버튼에 이주 가능 "!" 배지 + 새 거처 발견 알림
- 인게임 💾 저장 버튼 (저장 → 타이틀 → 내보내기 흐름 완성)
- 이주 시 "가구 N개는 이 거처에 남습니다" 확인, 성공률 50% 미만 출발 확인, 가전 회수 시 효과 중단 안내
- 이벤트 "⌄ 잠시 내려두기": 하단 칩으로 접어두고 자원 확인 후 다시 열기 (선택지 유지)
- 탐험 연속 실패 보정: 표기 확률은 그대로, 내부적으로 불운 연타를 완화

**연출/비주얼**
- 스탠드 조명: 켜면 갓이 안에서부터 빛나고 광원 헤일로 표시 (모든 발광 가구 공통)
- 쨍한 낮: 창가로 빛기둥이 떨어지고 먼지가 부유 — 궂은 날은 광량이 깎여 날씨 대비 명확
- 수첩/쪽지 종이 텍스처 리워크: 괘선·접힘 자국·세월 얼룩 있는 낡은 노트 종이

## v0.9.1 Beta (2026-07-04) — 베타 피드백 핫픽스

**치명 수정**
- 타이틀에서 새 게임/불러오기 창이 투명하게 떠서 아무것도 안 보이던 버그 (전 플랫폼)
- 백그라운드 탭/최소화 상태로 게임을 열면 시간이 완전히 멈추던 버그

**게임플레이**
- 식품 이원화: 🍎 신선식품(냉장고 없으면 부패) / 🥫 통조림(절대 안 상함), 섭취는 신선 우선
- 전반 수치 완화: 배고픔·갈증 소모 22%↓, 일일 청결 감소 절반, 탐험 최소 보상 +1, 양초 소모 2일당 1개
- 하루 정산 시 자동으로 에너지가 회복되고 다음날 아침 08:00부터 시작
- 자동 진행 모드 (Day 10 해금): 매 정시마다 치료→청소→최적 지역 탐험을 자동 수행
- 인카운터 등장률 대폭 상향(하루 60%), 탐험 도중에도 10% 확률로 사건 발생
- 고양이: Day 9부터 15% 확률로 단 한 번 찾아옴(놓치면 끝) / 입양 후 3일마다 음식 1개 소모, 굶기면 그날 애정 보너스 정지
- 일시정지 중에는 탐험·제작·취침 등 모든 행동 불가 (관람만 가능)

**UI/UX**
- 전체 UI 15% 확대 (폰트 가독성)
- 타이틀 화면에 💾 세이브 내보내기 / 📂 가져오기 / ⚙️ 설정 버튼 추가
- 설정 진입 정리: 모바일 = 우측 상단 ⚙️ / PC = ESC 키
- 패널이 겹칠 때 아래 창의 슬라이더가 위 창을 뚫고 보이던 문제 수정 (클릭한 창이 맨 앞으로)
- 퀘스트형 온보딩: 첫 이틀을 화면의 할 일 카드가 단계별로 안내 (수첩 안 읽어도 됨)

**사운드**
- 초 타는 소리 제거 (난롯불만 유지), 초를 켜고 끌 때 전용 사운드 추가
- 가구 클릭 시 가벼운 상호작용음

**모바일**
- 상태바 완전 숨김 (풀스크린 몰입 모드)
- 앱을 백그라운드로 보내면 음악 자동 정지, 복귀 시 재개
