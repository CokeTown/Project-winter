## 1. 최종 목표 (정본 문서 기준)

Steam 정식 출시 Base $9.99(론치 -15%≈$8.49)에 유료는 순수 코스메틱 Supporter Pack $3.99와 OST $4.99뿐, 확장 1.1~1.4와 2.0은 전부 무료 내장이다(SKU-PLAN §0~1, MILESTONES "출시=1.4 전량 내장" 프레임). 경로는 10월 Next Fest(데모 별도 AppID) → 1.0 정식 출시 → 1.1~1.4 월 1회 무료 업데이트 → 발매 1~2개월 후 2.0 「응답」 최종 업데이트(GD-2.0 서문, MILESTONES §3). 2.0의 비전은 콘텐츠 확장이 아니라 "이야기를 끝내는 업데이트" — 1.4에서 허공에 보낸 무전 송출에 응답이 돌아오고, 세관 너머 동부 신영토(사실상 본편의 2편: 셸터 4+지역 8)와 낙진 시계·연구실의 진실·엔딩 3분기(탈출/신세계/안식)+히든 「침묵」으로 서사를 완성한다(GD-2.0 §1·§2·§5·§6). 즉 1.x=세계의 완성, 2.0=서사의 완성이며, 무료 2.0 완료 시 "사실상 본편 분량"이 된다는 논리가 $9.99 장기값의 근거다(GD-2.0 §1, SKU-PLAN §1). 출시 게이트는 위시리스트 목표선·전 모드 오디트 그린·ko/en/ja·Steam Deck·클라우드 세이브·출시 트레일러가 전부 그린일 것(MILESTONES M0-d).

### 디자인 필러 (변하지 않은 축)

- 둘러싸였으나 혼자(Alone, Together) — 폐허=현대적 고립의 은유. 타인 접촉은 접촉 스펙트럼(부재/흔적~찰나의 긴장) 안에서만, 커뮤니티·우정·동거인 절대 금지, 유일한 동반자=고양이 (WORLDVIEW §2~3)
- push 금지·pull만 — '안 가면 죽음'이 아니라 '그곳에만 있는 갖고 싶은 것'으로 끈다. 기본 지역만으로 영구 생존 가능 유지, 코스매틱은 파워 0 (REWARD-LOOP §2·§4, GD-2.0 §6)
- 사망 없음·오프스크린 — '사망 없음, 탈진만' 정체성, 온스크린 전투 시스템 금지(발사=오프스크린 판정), 크리처 호러 금지·존(Zone) 문법 (GD-2.0 §3·§9.3, WORLDVIEW §3)
- 깊이='다시 마주칠 때 더 보인다' — 낙진 시계가 지도 자체를 바꾸고(3겨울 후 개방), 지역 숙련·대한파 프론트·부상 서사화가 재방문을 깊게 한다 (GD-2.0 §2·§9.4)
- 꾸미기=종착 재미이자 보상 엔진 — 가방(플로어)→지역 숙련→시그니처 코스매틱→꾸미기로 닫히는 pull 루프, 도시 시그니처는 복장 축 (REWARD-LOOP §3, GD-2.0 §6.4)
- 엔딩은 누적된 하루가 빚는다 — 어디서 겨울을 보냈나·무엇을 주웠나·무엇을 지었나가 leaning을 가중하되 강제 안 함(3택+보류 항상 개방), 갑툭튀 선택지 금지 (GD-2.0 §5·§9.8.8)

### 로드맵 골격

원칙: 출시 트랙 상시 우선, 2.0은 블로커 없을 때만 소비하는 선제작(비축) 트랙 (MILESTONES 서문). M0 정식 출시 준비=크리티컬 패스(a 스팀 파이프: 상점 제출이 최상류 의존성·캡슐·depot / b 데모 완성: 넥페 출품·ja 초벌·itch EN / c 마케팅: 트레일러 정본·페스티벌·텔레메트리 / d 출시 게이트 전부 그린) → Next Fest 10월 → 1.0 정식 출시. M1 기술 부채(#73 모듈화 Tier4·#114 문자열 외부화)는 M0와 병행, 2.0 전 정리. 출시 후 패치 사이클로 M2 2.0-α 「관문이 열린다」(세관 관문+항구도시 지역 2+셸터 1, 낙진 시계 연동) → M3 2.0-β 「도시로 깊이」(역 대합실·다리 관리소·지역 4·시그니처 복장 1차, 적대 다이얼·숙련 동부 확장) → M4 2.0 정식 「응답의 실체」(펜트하우스 종점·지역 8 완성·동부 서사 「응답」·엔딩 확장·#77 차기 도시 비축). 각 2.0 단계는 독립 패치 발행, 2.0-α는 넥페 출품 완료 후 착수 (MILESTONES §2~3).

### 기획 단계에서 기록된 폐기·피벗

- 기존 10지역→8 폴드 재편안 폐기 — 4도시는 기존 지도 재편이 아니라 동쪽 신규 영토('본편의 2편')로 피벗. 현 지도 불변, 파밍 8곳 전부 신규, 최대 리스크였던 첫 도시 sim 재캘리브 통째 소멸 (GD-2.0 §6 디렉터 정정 2026-07-09 2차·§9.8.3)
- currentCity 저장 필드 폐기 — 도시는 저장하지 않고 cityOf 파생으로, 구 §9.8이 지목한 세이브 대변경 리스크 회피 (GD-2.0 §9.8.1)
- 지역 숙련 revisitDecay(감쇠) 비채택 — '감쇠=징벌=코지 적대'. 떠남의 무게는 파괴가 아니라 '두고 온 것이 기다림' (GD-2.0 §9.8.6)
- 전파망 자유입력 비채택 — 욕설필터·신고·수동검수 부담이 코지에 과함. 큐레이션 문구 조각(무전 코드북) 조합형으로, 실시간 멀티 아닌 비동기 유리병 편지 (GD-2.0 §6.5)
- 구 엔딩 3분기 정의(연결/초월/영원회귀) → 디렉터 시나리오(탈출/신세계/안식+히든 침묵)로 재정의, Day10000 기존 엔딩은 '먼 에필로그'로 격하 (GD-2.0 §5·§9.5)
- 도시 '표시 그룹핑'(내부 4테이블을 2라벨로 가리기) 폐기 — '화장이 된다' (GD-2.0 §9.8.3, 이후 폴드안 자체도 폐기)
- Base 정가 $11.49 폐기 → $9.99 확정 — 적대적 리뷰가 $11.49를 'idle 밴드 초과+얇은 볼륨' 데드존으로 판정, 무료 2.0 완료 시 본편 분량 논리로 $9.99 장기값 (SKU-PLAN §1, 2026-07-12)
- 출시 시점 유료 확장 DLC 지양 + 별도 '에디션 SKU' 금지 — 유료는 코스메틱 서포터팩·OST 둘뿐, 에디션은 번들로 구현 (SKU-PLAN 원칙·§4·§8)
- push 설계 비채택 — 계절 창은 pull 수정자일 뿐, '겨울 오기 전 떠나라'로 미끄러지지 않게 생존은 절대 게이트하지 않는다 (REWARD-LOOP §2, GD-2.0 §9.8.5)
- 대한파 프론트의 sim 편입 재도입 금지 + coldSnapForecast 제거 불가침 — 지우면 하드코어 치명성 붕괴(사망 25%→5%, 20시드 실측) (GD-2.0 §9.4)
- '완전한 고독' 톤 → '둘러싸였으나 혼자'로 세계관 피벗 — 기존 대면 인카운터를 캐논 안으로 흡수, 아트/인카운터 재작업 0 (WORLDVIEW 서문·§4)
