# 트레일러 스크립트 정본 v3 (디렉터 스크립트 2026-07-09 — 「촛불에서 아홉 번째 겨울까지」)

> 자막·라이터 소리·OST는 **전부 디렉터가 편집에서** 얹는다 — 에디션은 그림과 SFX만 담당.
> 원칙 유지: Frostpunk의 반대 — 내레이션 없음, 오케스트라 없음, 사망자 카운터 없음.
> 리듬: 비트 간 암전 0.25초(구 0.7초 — "끊긴다" 지적 해소), 셸터 투어는 무암전 하드컷.
> 이 문서가 `src/trailer-script.js` beats 배열의 정본이다.

## 비트 시트

| # | 컷 | 길이 | 내용 | 자막(디렉터) |
|---|---|---|---|---|
| B1 | 촛불 점화 | ~6s | 전 광원 소등의 어둠(라이터 소리 자리 1.6s) → **촛불 하나만 켜지며** 방이 드러남, 살짝 풀백 | — |
| B2 | 꾸미기 스톱모션 | ~7s | 빈 옥탑방에 가구 9점이 0.6s 간격으로 하나씩 놓임 → 완성 홀드 | Decorate ur Own Shelter |
| B3 | 전리품 | ~8s | 정산 스태거 개봉 + 잭팟 팝업(유지) + **획득창 희귀 하이라이트: 도료=빛나는 금색 / 도면=핑크 글로우 배지**(펄스 애니메이션) | With What you loot |
| B4 | 전 셸터 투어 | ~13.5s | 1.4 기준 **12셸터 전부** 꾸며진 채 컷당 ~1.1s 하드컷, 시간대 10:00→22:30(낮→밤), 컷마다 서서히 줌아웃 | where ever you want |
| B5 | 해와 날씨 | ~10s | 와이드 고정 — 05:30→21:00 연속 스윕(해가 뜨고 진다) + 날씨 맑음→비→눈 순차 전환 | To survive, To live, To Rest |
| B6 | 고양이 | ~7s | 미디엄 샷(실좌표 팬) → **인게임 "클릭 시"의 그 확대 클로즈업으로 전환** | With your Lovely Friend |
| B7 | 엔딩 홀드 | ~7s | 눈 내리는 밤의 완성된 집, 아주 느린 접근 → 홀드 | Until 9 winters pass. |

- 투어 순서: container → bus → cabin → rooftop → greenhouse → subway → ship → tugboat → lighthouse → controltower → lodge → bunker (여정감: 시작의 컨테이너에서 최종 벙커까지).
- 로고 오버레이는 v3에서 제거 — 자막 흐름과 겹치므로 편집 몫.
- 오디오: 에디션 BGM 뮤트 고정, SFX 유지. B1 라이터 소리는 편집 삽입(인게임 소스 없음).
- 희귀 하이라이트 배지는 에디션 전용 주입 — 본 게임 정산 UI 정식 반영은 별도 백로그(데모/본선 공용 개선 후보).
