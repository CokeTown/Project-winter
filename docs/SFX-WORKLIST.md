# SFX 수요 워크리스트 (디렉터 소싱용, 2026-07-23)

> 오더: "중복 사용 SFX 분리 — 탐험 등 행동 전용, 가구 재질별(천/플라스틱 등), 인카운터 전용(현 전파 노이즈 말고)".
> 아래 표의 **파일명 그대로** 주시면 드랍 즉시 배선합니다(태깅·호출부 교체·게이트는 Fable 몫).

## 0. 납품 규격

- 포맷: **WAV/mp3/ogg 아무거나** — 변환은 `tools/gen-sfx.ps1`(ffmpeg)이 처리.
- 길이: 액션음 0.15~1.2s · 인카운터 스팅 1~2.5s. **루프 없음**, 모노 무방.
- 볼륨: 인게임 마스터가 7/100로 낮게 깔리므로 피크 -1dB 정도로 평준화만. 잔향 짧게(방치형 — 소리가 공간을 오래 점유하면 피로).
- 드랍 위치: 아무 폴더나 전달만 — 이후 `public/sfx/`로 규격 변환해 배치.

## 1. 중복 실태 (코드 전수 실측)

| 현재 파일 | 사용处 | 실제로 퉁치고 있는 상황 |
|---|---|---|
| `craft.ogg` | **20곳** | 가구 제작·가방 제작·**개조/프로젝트 완공**(옥상 텃밭·벙커 지붕·지하철 허브)·**방호복/총 수리**·염장·**도색/벽지/테마 적용**·**지식 해금**·가구 티어 손질·발견 배너·상자 개봉 연출 — 전부 같은 망치질 |
| `place.ogg` | 7곳 | 가구 배치 확정 + 가구 선택 '톡' + 상호작용 '톡'(피치만 변조) + 퀘스트 완료 플래시 |
| `whoosh.ogg` | 4곳 | 가구 수거/인벤 수납 + **옷 갈아입기**(주석으로 이미 "전용 cloth 소스 오면 교체" 예약됨) + 모달 전환 |
| `door.ogg` | 3곳 | **탐험 출발**과 문 여닫이가 같은 소리 |
| `sting.ogg` | 1소스 | **인카운터 등장 전량** — 피치 변조로 3톤(narrative 1.0 / danger 0.82 / warmth 1.18) 흉내 |
| 청소 | **무음** | cleanShelter에 효과음 배선 자체가 없음 |

## 2. 필요 슬롯 — P1 (13파일이면 체감 격변)

### A. 행동 전용 (5)

| 파일명 | 쓰일 곳 (현재) | 소리 방향 |
|---|---|---|
| `act_depart.ogg` | 탐험 출발 (현 door 재탕) | 배낭 들쳐 매는 부스럭 + 눈 밟는 첫걸음 한둘 |
| `act_clean.ogg` | 청소 (현재 무음) | 빗자루 или 걸레질 두어 스트로크 |
| `act_outfit.ogg` | 옷 갈아입기 (현 whoosh — 예약된 그 건) | 천 스치는 스윽, 지퍼 살짝 |
| `act_migrate.ogg` | 이주 확정 (현 whoosh/door) | 짐 끌기 + 문 닫힘 여운 |
| `act_book.ogg` | 지식 해금·책 관련 (현 craft) | 책장 넘김 + 탁 덮기 |

### B. 가구 재질별 배치음 (5) — 오더 핵심

배선 설계: 가구 DEFS에 `sndMat: 'wood'|'cloth'|'metal'|'glass'|'paper'` 태그를 박고
배치 확정음만 재질로 갈라냅니다(선택/상호작용 '톡'은 현 place 유지 — 그건 UI 소리가 맞음).
50종 분류(태깅)는 Fable이 합니다.

| 파일명 | 대표 가구 | 소리 방향 |
|---|---|---|
| `place_wood.ogg` | 침대·테이블·책장·의자·캣타워 등 대다수 | 나무 바닥에 나무 다리 툭 |
| `place_cloth.ogg` | 러그·방석·커튼·소파·해먹 | 천 툭 + 눌림 퍽 |
| `place_metal.ogg` | 난로·발전기·정수기·냉장고·LED 바 | 금속 무게감 쿵 + 짧은 떨림 |
| `place_glass.ogg` | 거울·액자·랜턴·무드등 | 유리/도기 조심스런 탁 |
| `place_paper.ogg` | 책 무더기·골판지류(캣타워 T1) | 종이/골판지 부스럭 툭 |

*(선택) 수거 2종*: `pick_soft.ogg`(천·종이) / `pick_hard.ogg`(나무·금속·유리) — 현 whoosh 대체. 풀 5재질까진 불요.

### C. 인카운터 전용 스팅 (3) — "전파 노이즈 말고"

현 sting.ogg 1개를 피치로 늘려 쓰는 3톤을 실소스 3종으로 교체합니다.
방문자 라디오 대사(radio_broadcast)와 겹치지 않게 **전파/노이즈 계열 금지** — 악기성 스팅으로.

| 파일명 | 톤 (코드 분류 실존) | 소리 방향 |
|---|---|---|
| `sting_narrative.ogg` | narrative — 서사·중립 이벤트 | 낮은 피아노/현 2음, 여운 짧게 |
| `sting_danger.ogg` | danger — 위기·도둑·위험 선택지 | 낮은 불협 현 스팅, 위협적이되 호러 금지 |
| `sting_warmth.ogg` | warmth — 선물·방문자·훈훈 | 밝은 3음 상승, 따뜻하게 |

## 4. 유지(교체 불요)

craft.ogg(진짜 제작 전용으로 축소 잔류) · place.ogg(UI '톡' 전용) · ring(업적) · purr/meow(고양이) ·
sting.ogg(3종 도착 시 폐기) · 앰비언스/빗소리/동물(wl_*) 전 계열.

## 5. 도착 후 배선 순서 (Fable)

① gen-sfx.ps1 잡 추가·ogg 변환 → ② DEFS 50종 sndMat 태깅 → ③ 호출부 교체(craft 20곳 분해 포함)
→ ④ 하네스 dbgSfx 채널로 트리거 전수 확인 + 배터리 → ⑤ 데모 재수렴 시 동반 유입.
