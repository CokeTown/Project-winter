# Project Shelter — 웹 기반 생존 기지 꾸미기 (v1.0)

## v0.8 ~ v1.0 신규

- **생존 게이지**: 🥫배고픔/💧갈증 — 시간·탐험으로 소모, 게이지 클릭 또는 자동 섭취로 회복. 25 미만 탐험 -10%p, 0이면 탈진(탐험 불가). 사망은 없습니다 (Cozy Survival).
- **도트 지도 탐험**: [🎒 탐험] → 절차 생성 도트 지도에서 파밍 지역 선택. 현재 거주 구역과의 **거리만큼 소요 시간이 달라집니다** (0.8x~1.35x).
- **계절**: 12게임일 = 1계절 (🌸→☀️→🍂→❄️). 계절별 환경 색조·날씨 편향, 겨울엔 잔설이 남고 배가 빨리 꺼지며 온실 텃밭이 얼어붙습니다. LCD 시계에 계절 표시.
- 모바일 BGM 재생 수정.

포스트 아포칼립스 셸터 게임 **Project_Shelter**의 웹(Three.js) MVP입니다.
핵심: **"꾸민 셸터는 예쁜 배경이 아니라 플레이어를 살리는 생명 유지 장치다."**

## 실행

1. `index.html` 더블클릭 (Chrome/Edge, 인터넷 필요 — Three.js CDN)
2. 또는 `powershell -ExecutionPolicy Bypass -File serve.ps1` → `http://localhost:8420`
3. **모바일 지원**: 터치로 전부 조작 가능 (탭 선택/배치, 드래그 이동, 빈 곳 드래그 = 시점 회전, 두 손가락 핀치 = 줌+회전)

## UI (v0.6)

- **모든 패널 이동/접기 가능** — 상단 손잡이(⠿)를 드래그해 원하는 곳에 배치, `–` 버튼으로 접기. 위치는 자동 저장.
- **전자시계(LCD)** — DAY·시각·시간대·날씨를 표시. 깜빡이는 콜론까지.
- **온스크린 카메라 컨트롤** — ⟲⟳ 회전, ＋－ 줌, ⌂ 시점 초기화 (모바일 필수, 데스크톱도 편리)
- 배치 중에는 **✖ 배치 취소** 버튼 표시 (모바일에서 Esc 대체)

## 코어 루프

탐험 준비(준비물 6종) → 파밍(자원 11종) → 부상/치료 → 꾸미기·유지(조명 연료, 청결, 유지비) → **제작** → 하루 결산 → 이주

- 시간: 하루 ≈ 16분, 새벽→밤 조명 변화. 날씨는 게임 1~2일 단위.
- **제작대(🔨)**: 천→붕대, 부품→건축재, 부품+배터리→조명, 그리고 정수기/발전기/냉장고 제작.
- **가전 시스템**: 🧊냉장고(음식 부패 방지) · 🚰정수기(매일 물+1) · ⚡발전기(연료 1/일로 그날 배터리 소비 무료). 없으면 매일 음식 1개가 상합니다.
- **📻 라디오 예보**: 라디오를 배치하면 내일 날씨를 알 수 있습니다 (등대는 기본 제공).

## 구역과 이주 (v0.7)

월드는 5개 구역으로 나뉘고, 각 구역에 셸터들이 속합니다. 구역 위치는 탐험 접근성에 영향을 줍니다.

| 구역 | 셸터 | 위치 보너스 |
| --- | --- | --- |
| 🏜️ 잿빛 외곽 | 컨테이너 · 스쿨버스 | 주거지역 +3%p |
| 🏙️ 무너진 도심 | 옥탑방 · 지하철 | 상업·슬럼 +5%p |
| 🌾 초원 구릉지 | 돔 벙커 · 온실 | 주거지역 +5%p |
| 🌲 숲과 산기슭 | 오두막 | 공업지대 +5%p |
| 🌊 잿빛 해안 | 여객선 · 등대 | 슬럼가 +5%p |

**이주 난이도**: 처음 입주하는 셸터는 **정비 자원**(예: 벙커 건축재2+배터리1, 등대 연료2+부품3)이 들고, **다른 구역으로의 이주**는 추가로 음식1+물1과 게임 시간 3시간이 소요됩니다. 같은 구역 내 정비된 셸터끼리는 자유 왕래.

**셸터별 현실 제약**:
- 📦 컨테이너: 얇은 철판 — 비/눈 오는 날 쾌적함 -8
- 🛖 벙커: 전력 끊기면 보너스·특성 정지
- 🏙️ 옥탑방: 노천 — 비/눈마다 청결 -3
- 🏡 오두막: 목조 지붕 — 악천후엔 매일 건축재 1 (없으면 누수, 청결 -8)
- 🚌 스쿨버스: 좁음 — 가구 최대 8개
- 🚇 지하철: 완전한 어둠 — 켜진 조명 없으면 쾌적함 -12
- 🌿 온실: 유리 지붕 — 눈엔 건축재 1 보수
- 🚢 여객선: 습기 — 청결 매일 -2 추가
- 🗼 등대: **옥상 빗물받이** — 비/눈 오는 날 물 +2 (텃밭·낚시처럼 파밍 없이 자급 가능). 전망실은 실내 거실로 개편
