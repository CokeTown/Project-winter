# Project Shelter — 웹 기반 생존 기지 꾸미기 MVP (v0.6)

포스트 아포칼립스 셸터 게임 **Project_Shelter**의 웹(Three.js) MVP입니다.
핵심: **"꾸민 셸터는 예쁜 배경이 아니라 플레이어를 살리는 생명 유지 장치다."**

## 실행

1. `index.html` 더블클릭 (Chrome/Edge, 인터넷 필요 — Three.js CDN)
2. 또는 `powershell -ExecutionPolicy Bypass -File serve.ps1` → `http://localhost:8420`
3. **모바일 지원**: 터치로 전부 조작 가능 (탭 선택/배치, 드래그 이동, 빈 곳 드래그 = 시점 회전, 두 손가락 핀치 = 줌+회전)

## UI (v0.6)

- **모든 패널 이동/접기 가능** — 상단 손잡이(⠿)를 드래그해 원하는 곳에 배치, `–` 버튼으로 접기. 위치는 자동 저장.
- **전자시계(LCD)** — DAY·시각·시간대·날씨를 표시. 깜빡이는 콜론까지.
- **온스크린 카메라 컨트롤** — ⟲⟳ 회전, ＋－ 줌, ⌂ 시점 초기화 (모바일 필수, 데스크톱도 편리)
- 배치 중에는 **✖ 배치 취소** 버튼 표시 (모바일에서 Esc 대체)

## 코어 루프

탐험 준비(준비물 6종) → 파밍(자원 11종) → 부상/치료 → 꾸미기·유지(조명 연료, 청결, 유지비) → **제작** → 하루 결산 → 이주

- 시간: 하루 ≈ 16분, 새벽→밤 조명 변화. 날씨는 게임 1~2일 단위.
- **제작대(🔨)**: 천→붕대, 부품→건축재, 부품+배터리→조명, 그리고 정수기/발전기/냉장고 제작.
- **가전 시스템**: 🧊냉장고(음식 부패 방지) · 🚰정수기(매일 물+1) · ⚡발전기(연료 1/일로 그날 배터리 소비 무료). 없으면 매일 음식 1개가 상합니다.
- **📻 라디오 예보**: 라디오를 배치하면 내일 날씨를 알 수 있습니다 (등대는 기본 제공).
